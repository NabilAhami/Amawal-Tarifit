 uïïun 130, sinyur 2958 (Février 2008) Tamazight Tanfust taqudvadvt: Sghin imar nnegh! Yura t: Said BELGHARBI Ayt n ttareboin ugin ad zenzen aman n tara nsen. Ajemmadvin din ij n ubagher itehrara s wafriwen nnes x ijj n umardvus n ijj n utarras ijeyyef it din razv. bagher yedjuz yufa tyawant nnes deg weyysum n umettin nni…. Tiriw tiqqest deg urawen n imezdagh, yehsvar uzder deg wehsin nsen. Ikkar ubarrah deg umalas n ssuq ameqran: -Aya yri n rbayoet, a ayet bab n tudart, xarsvem cwayt deg inni d awem d yudsen, tmettan s fad kenniw tara nwem tessawem zzay s tazeggûrt, jjet aman n tareywin n tarawin nwem tazrent d irizazen x fad nsen di yezzughmet bupeyyuf nwem ibriden ghar tyawant… Ayt n ttareboin ismedv asen umnus deg ifassen, izziyzu day sen utebpet x wudem awen nsen. Ma tara nsen ad t zzenzen nigh ad t bdvan aked imezdagh n ujemmadvin. Sghuyyent tefruxin n dcar: -Upu!!! Tara war tedji n bettvu, war tedji n midden, tara n ayt bab nnes, ayt bab nnes d neccin. Sghin idvennadv nnegh, sghin imar nnegh, itghima yasen mghar tara deg uraq nnegh war tesghin, tehraran xaf s mammec itehraran ibaghriwin x inighiten n razv d fad. (17-11-2007) 2002 Tawiza. .
