 uïïun 175, ymrayur 2961 ( 2011) Tamazight tamssizwarut: mulay hicam issavul d "ddahir lbarbari" (Le prince Moulay Hicham réinventer le "Dahir berbère") yura t: mupmmd budhan addad bnid i tmazivt, i issnfla (exprimer) mulay hicam dg umsawal nns akd unvmas (journaliste) stifn smit dg tsvunt "Débat" uïïun n 166 n iyurn tçayur d mrayur (, octobre) 2011, d amayuj (dangereux) aïïas acku bab n waddad ad d yan n waggag (intellectuel) itari dg tsvunin d timaäalin, arni is ntta d agldun d mmis n twacult tagldant i nbbän dg lmrruk. asuvl n umyey n "ddahir lbarbari": asmi issqsa "stifn smit" mulay hicam xf tnbaät n mupmmd wisn säis di tmnäawt tamaynut, yarra is tinbaäin nns ur zzaysnt iktim (diminuer) walu. maca ikmma (porter atteinte) tayunt tanamurt (unité nationale) umi issikz s tmazivt di tmnäawt. manayad d avul vur "ddahir lbarbari" i tgga fransa puma ad tbäa jar imrrukiyn. asmnäw (constitutionnalisation) n tmzivt d tutlayt u d tanttit (identité), issuncuk, dix, am itwara mulay hicam, yat n tkmmawt (atteinte) i tayunt tanamurt. asmi vad nvaë awal ad, ad nraja (nggan) tivëi n "llaïif" i tuva itwavaën akd "ddahir lbarbari". mulay hicam tuva days "taryazt" acku ur ibdd vur tumayt (contenu) waha n "ddahir lbarbari" mamc tggn ass ad qao imazivufubiyn imaynutn, ntta yidar "ddahir lbarbari" s izwl (nom) d tumayn nns munen. da ätaran d xaf nv aïïas n isqsan: -min ijjin yan n waggag d atrar (moderniste) i tddar di timiäi (siècle) 21 ad ixzzar var tmazivt zg isrkisn i d iffvn di 1930, isrkisn nni (nna) zi iyuncuk umyey (mythe) n "ddahir lbaebari"? -mamc imsar alami mulay hicam itfulla (s'appuyer) xf isrkisn zund ca n tidtt ur digs turda? astan (réponse) is "ddahir lbarbari" oad iqqim d asavul (au sens de "référence") n uswingm d uxaëës vur waïïas n waggagn imrrukiyn. max "ddahir lbarbari" oad itavd (orienter) aswingm n waggagn? acku ntta d yan n umyey d amdaddt (véritable). man wn issavuln "ddahir lbarbari"? asmi issqsa mulay hicam ixf nns: "is d nkni vad d issavuln "ddahir lbarbari" n 1930 i d yusin ad ibäa jar imrrukiyn?", ur d as d iwäi xf twngint nns is, s usqsi yad, ntta issavul d s tidtt "ddahir lbarbari". asuvl n tidtt i "ddahir lbarbari" maci d asikz zi tmazivt, illan di tidtt d tamnviwt n "ddahir" ad, asuvl nns d twagit n usikz n tmazivt u d tagdlt (exclusion) nns mamc itgg mulay hicam. tumayt n "ddahir lbarbari" d abddi dg udm n tmazivt puma ur vur s itli wansa nns i tnzgga (mériter). tuva ntggan zi mulay hicam, mc ntta d uvdim (juste, équitable) arndad (bnid) i tmazivt, iri istl (izvn = critiquer) agllid mupammd wis säis xf usikz amnnurf (incomplet) n tmazovt, tammaskalt (au lieu) n tardayt (accusation) nns s usuvl n "ddahir nbarbari" i ikks di tidtt s usmnäaw n tmazivt. imyabatn n mulay hicam imala "ddahir lbarbari" tuva yuss d ad inäa jar imrrukiyn, mamc inna mulay hicam, iqqn ur itettu is d imiru (grand-père) nns mupammd wis smmuã i t izmln (signé) s ufus nns. imala asikz zi tmazivt izdda (menacer) tayunt tanamurt, is "lbulisaryu" itettarn ad bäan lmrruk jar sin n iwunak, ggin manaya s tmazivt nvd s usuvl n "ddahir lbarbari"? maci ggin manaya s usavul nsn aorab u maci amaziv? 2002 Tawiza. .
