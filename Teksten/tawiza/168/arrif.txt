Zdat i tsimant4 n Lmerrvuk deg 1956, tugha Arrif d ict n tmurt iscan ghar s Lmexzen s yisem n blad ssiba, di twarat d icten blad lmexzen tugha tt d tamurt i xef yesdurri Ssultvan. Tumliyin n isental5 a kint id z ghar reboadv n inmezrayen6 d irumiyen i d yeswizvden ghar tmesrayt a mahend ad agh innin belli: Arrif netta oemmares war xaf s isdurri Lmexzen, Arrif netta tugha t d tiqebbar i war ixisen ad irint deg ufus n uzviyyar, Arrif d aydud i tugha yedjan mgal i marra asdurri, ammu s isaran7 n inmezrayen a nttessen ra d min tugha yeona Lmexzen ameghrabi adaxer igudar n tghermatin8 nnes, d mayemmi ira tugha ikkessi isem n blad ssiba d ijj n uzawar i marra tiqebbar d redcurat d timizar i war izemmar ad yadef deg wakud9 nni yeodan, tiqebbar i ntaf ghar unegaru ya aqa tent marra adaxer i yejj n wudem, udem n Tegldit tamaghrabit10. D.M.Hart deg wawar nnes yenna: Tamurt n Lmexzen tugha tt d ict n rudva tusi d xef tghart n Atlantik11 deg ughezdis n waday n Lmerrvuk, suya tamurt n Lmexzen tugha tt, war tiywiydv ura d azyen n Lmervrvuk i yedjan ruxa. Di twarat d icten tamurt n ssiba tugha tt d ict n tmurt d tameqrant attvasv, s idurar d tinarzufin, maca Lmexzen netta ira itragha ghar s ismawen war yecnin, amacnaw: Tamurt n Imenghan, Tamurt n Ssiba nigh Tamurt n Imenzagh12 (Part, 1976:8).
