 uïïun 152, mggyur 2959 (Décembre 2009) Tamazight Mayemmi neccin d imeggura? Par: Andich idir Tsuffegh d tanbadvt tamerrukit ijjen n wezref d amzazzvar (raciste) iteggi hima ad yejj ighermanen (citoyens) imarrukiyen ad semman tarwa nsen s ismawen d imazighen id asen farnen, qqaren imasayen iorubiyen n tmurt nnegh belli ismawen imazighen ssehcamen u war hlin s minzi zugguzen (expliquer) tutlayt tamazight s taorabt amecnaw addad n isem «ayur», (xzvart taqubanit n imasayen nnegh ar mani tiwedv!). Di tidett tanbadvt tamerrukit nettat umi ixessv ad tmdvha (ad tssdha) zeg ixf nnes, minzi tekkes ismawen n tarwa n tmurt i ghar yella ijjen n wammak (signification) d asebhan u yecna s wattvas. U tezzvu deg udghar nsen ismwen d infuren udfen d aked uhewwes Aorab i Tamezgha, ujar zi manawya mara nssen min xsen ad inin cra zzay sen am (Mapa = tafunast), Pind = 100 n tleghmin nigh dinni cra n ismawen illan mgal i yejjed n lislam amecnaw Oabdnnbi… Ad nssarwes ddeg a n tsertitin timzzazvarin iteggen iorubiyyen ifaciyen s labartvayed i tugha tdeffar tadersi (minorité) n icmlalen di tsghart n ticcemt (majorité) taberkant di yeffus n Tefriqt (ifriqeya). Ammu nican itmsaren di tmazirt nnegh, tadersi n ifasiyen harren xef idgharen deg uzellif n tnbadvt, teggen min xsen deg ijjen waydud ticcemt nes d aqubbani. Marra yenni iksin amnus n izerfan n ufgan nigh imeghnas itarran taynit i tghawsa yad oeyben labartvayd amzazzar amerruki. Tammunt (évidence) xef wawal inu d tabrat i tsekk Pyuman Rayts Wutc ( Human Watch) i wemghar n ughlif n ugnsu Cakib Ben Musa tettar day s hima ad yaggêj zeg usbeddi nnes i ysmawen imazighen. Ixesv anegh ad nssen belli aonkrif n ismawen imazighen netta d ijjen n wudem d amezvyan zeg wudmawen n tamazighufubit itegg uwnak n umerruk. Danita iqqen ad nidvar tukardva n lmerruk i tmura n imekraz imazighen iteddaren di atvlasv afad ad ten tsbezzizen xef uzawag ghar timnadvin issawlen taorabt bac ad ten settun di tanettit nsen tamazight, arni xaf s askurmut n inelmaden di tsdawit n Imtghren, Ameknas … d uselyem n ukabar adimuqratvi amazigh amerruki snbghar ma nssiwel xef ustegh (réprimer) nnes i tismkanin (les manifestations) itmsarent di kuy taghmart deg wammur nnegh. Tisertitin a timazighufubin d ijjen n useddi d amezyan zeg ict n tseddect d tazirart d taorubit, iswi nnes assoarreb n ufgan d udghar afad ad yaghul d aorab. Marra manawya slmsnet cra n ifasiyen d inazviyen imddukal n uhewwas Afransis immexsen imazighen i lebda (umyey n ddvapir lbarbari, izerfan n usoarreb d ijjen umedyet …) skren ijjen n umezruy d amaynu itic asen tazrfanit i wnbadv u isettu deg inagan n tmurt nnegh n tidet (Mulay Muhend, Aqccic, Lmseodi, Azayay, Uobdslam, Memmis n rhaj Sellam…) Min dayi ijjen areobegh (tvbeptegh) marra yen immexsen tmazight d ifasiyen nigh tarran ixf nsen d ifasiyen am uhngar n ukabar uzarug (parti de listqlal) Oebbas Lfassi id ilulen di Berkan u netta yennan: (ad ghensegh mahend tmazight war ttili d tunusvibt) ammec yenna wenni i d as yezzaren oellal lfassi (ad xaf nnegh nbdven udayen ula nebdven xef nnegh ccluh …) Zi turda ynu mahend inbadven nnegh qqnen ixf n tmurt nnegh ghar taydyuljit taorubit oemmar s ad nggoed tsawent di marra igaren n tudert ar tunnunt aqat d taneggarut aked ughezdis n uselmed d tadvmsa d tadlsa u min d anegh yarennin tifqqeht Filistvin d Djibuti ghar sen tansa tggeod xef tenn i ghar s ghar tmurt nnegh deg yeyyar n uselmed. Marra ineqqisen (les rapports) n tmsmunin n izrefan n ufgan nigh n lbank amadval isrusa lmerruk deg umeggar. S tagedlt min zzun Aorben n umerruk qa mjjarn t id rexxu, mjarren d ixarriqen nsen d usoarreb nsen i imazighen. () 2002 Tawiza. .
