 uïïun 141, ynyur 2959 (Janvier 2009) Tamazight Reowin n tayri Tura t: Mayssa rachida EL MARRAKI Isudv d reowin imruda, reowin d areqagh d ahmaymi… ifsar atweseg, ifsar issiq nes marra mani mma, yus d ad yini ca yarzf d ad yarni ca di tudart n usefru. Reowin n tayri isudv d tiwarin isudv d taycit isudv d targhagh, id isifedv wezrawi negh id isqad d tarzift nigh d awardi i wmadval n idelisen, wanita i yarni ghar yiyyar n usefru yisi d akid s cra n wattvas d wattvas n tinawin, tikemicin n tesmaqarin tiourac n taycit sghemyent tayrireqhend deg utlao. Ssipa … ssipaaa yaghirin i waman s ujamadvin, manis d anagh d isifedv wezrawi nagh Krim KANUF "Reowin n tayri" nes d ijj n udelis arni ghar s CD: adelis a i d anegh d isughlen marra tifriwin n yitvij d terghegh n tayri, adelis a i di ntaf tifras n tayri n tmexsa s wattvas n wudmawen d wattvas n tebridin d wattvas n taycitin… "Reowin n tayri" irur d d amaynu yiwi d akid s amaynu yexs ad anegh yini attvas n umaynu di tiru nes. Nec war tighegh xef udelis a war sinegh ad aghegh, maca rami ghrigh srigh Reowin n tayri dwint id kra n tinawin daroent i tefray inu mahend ad inint reowin ad siwrent reowin ad hadant reowin… 206 isebtar, 31 n tqesisin urant s tlatinit urant s tifinagh, aodar d tamsezwarut n hassan benhakia d lhussin farhad, tasmaqalt n weoban lbachiri abd lkader, trifa graf i t ismunen d adelis tsufgh it id… atg D wanita d drus zeg wattvas zi tefras n "Reowin n tayri", reowin d asemadv reowin ihma reowin d areqagh reowin d acemrar… Reowin n tayri… Reowin n raxart… 2002 Tawiza. .
