 Uttvun 85, Semyur 2004 (Mai 2004) Tamazight Tamessizwarut: Max imughrabiyn tiffen innidven deg iyyar n usiwef islamawi? (Pourquoi les marocains surpassent les autres dans le domaine du terrorisme islamiste?) Yura t: Muhemmed Budpan D imughrabiyn i yenghin anazbay (résistant ="amenkar" aussi). D nitni d imasayen (responsables) s uslid (directement) xef iduqquzven (explosions) imssawafn (terroristes) n Madrid. D amughrabi Obdllap Tabarak i yessenjmen Ben Laden s ubbazv (arrestation) n imirikanyn deg idurar n Turabura mani yisi atilifun n ufus (téléphone mobile) n umghar nns Bn Ladn huma aneggaru yad ad irwel u ad itwattvef deg araq (ansa) nnes Oebdellap Tabarak. Amughrabi Abu Saoid Lmughribi yemmut di Ccican mani tugha itawz icicaniyn. D nitni i isduqzven Anfa (Ddar Lbidva) di 16 mayyu 2003. D nitni i tugha itannalen (soutenir) taghrasvt (cellule) n Pamburg (Hambourg) mani itwaghawes (être planifié) awwat n Umirikan di 11 Sibtumbre 2002. 17 n imughrabiyn llan d inkraf (prisonniers) di Gwantvana. Izewlen (noms) nsn ccurn Ikaramen (dossiers) n tarezzut xef tighrasvin n imessawafen deg ûmadval. Yaghul umughrabi d inirdi (accusé) al ami imirikaniyn sbedden ca n ibarlamaniyn deg unafag (aéroport) tghil asen d imessawafen acku d imughrabiyn. Isefka (données) yad mmalen (ccanen) tipawt (présence, "anegmet" aussi) taxatart n imughrabiyn deg usiwef amadval zeg ijj n wudem ttumanen (paraître) akid s d isatven (héros) n tuddsa (organisation) n "Lqaioda" n Ben Laden. Ayad isrusa xaf negh aseqsi yad: Mayemmi (max) dewlen (ghulen) imughrabiyn d isekkiwn (serviteurs, 'iqeddacen' aussi) imaglayn ujar (les plus fidèles serviteurs) n tadiyuluji tamessawaft n Bn Laden aorab? Di tidett, min (may) teggen imughrabiyn ad, itiffen igherfan (peuples) innidven (yadvnin) deg tsekkawt (service, 'tiqeddect') n usiwef, d netta s imant nnes i ytegg unabadv (gouvernement), ikabaren, timesmunin taoarabin, imdalsa (zeg 'idles' = intellectuels) imughrabiyn… Min (may) izvelyen (distinguer) qao tisentiwin (institutions) ad, ur id d advfar (dépendance, suivisme) nnes i wugmudv (orient) aorab ukan. Itezvlay itent usuki (exagération) deg udvfar nsen i qao min d itasen zeg ûoraben. Asuki yad ntaf it' deg ufugh s imilyunen (millions) n iwdan di timeskanin (manifestations) n umyawaz (solidarité) aked ifilistviniyn d ioiraqiyn. Aoraben n tidett teffghen s ca n timidviwin (centaines) wapa. Ntafi t' oawd deg usughul n Rbatv d tazuzzvent (capitale, 'tamaneght' aussi) n idles aorab. Ntafi t' deg usoarreb ughrid (arabisation totale) i zi tettaren ikabaren. Ass ad, aked ufsar (le fait de se répandre) n tumant (phénomène) n usiwef, imughrabiyn i yddvaffaren asiwef ad ur tegzan (se limiter) s usmemmes (zeg 'memmis' = adopter) d umzagh (défense) nnes. Taghulen d alen nnes, zund (am) min imsaren di Anfa, d Madrid. Wad d yan n wudem n usuki n imughrabiyn deg udvfar nsen i Ben Laden. Anezzugh (tendance) ad advnyan (maladif) gher usuki di tsekkawt n tghawsiwin n waoraben, yedwel (yaghul) d ict n tanumilt (caractéristique, attribut) n imughrabiyn awarni uzarug (indépendance), mani yaghul Lmughrib d yat n tmazirt d "taorabt". Manis d yeffegh unezzugh ad advyan? Man ammak (sens, signification) ghur s? Yeffegh d zi "Amussu aghelnaw" (Mouvement nationaliste) yeggin zi "tuoourba" n Lmughrib ijjed (religion) nnes amaynu. Uma ammak nnes amdaddet (véritable) d aseccen (le fait de montrer) is imughrabiyn ur llin d aoraben. Acku ssnen is ur llin d aoraben, s uyenni ssukkuyn (exagérer) deg tsikkawt nsen i tghawsiwin taorabin huma (afad) ad umanen d aoraben u ad zay sen ssikzen (reconnaître) aoraben imdaddaten. Mer (olik) llan d aoraben ili ticli (comportement) d waddad (attitude) nsen arendad (bnid) i tighawsiwin n waoraben igga zund ticli n waoraben s imant nnes, ili ur days bu asuki i ntaf ghar imughrabiyen. Asuki yad ticcen (tefkan) zzays imughrabiyen anza (preuve) xef "tuoourba" nsen ur ghar sen illan. Imala (si, 'iy', 'mer') inebbadven n Lmughrib ran (xsen) s tidett ad arnan (vaincre) asiwef, yuccel ad bbyen aked ugmudv aorab manis ur d anegh d yusi ghas (mghar) tiggenzvit (sous-développement) d usiwef. Asvurif amzwaru deg ubrid n tubbya aked waoraben d asikkez, d amendvaw (constitutionnel) d ughrid u d amzul (courageux, 'abghas' aussi), s tnttit (identité) tamazight n Lmughrib d usudvrer (déclaration) unsvib is Lmughrib d tageldit d tamazight (monarchie amazighe). Yuccel oawed asbeddi n usoarreb i yredlen (démolir, détruire) anli n tarwa nnegh u yarra ten d tifezwayin (bombes) sduqquzven tent iweppabiyn zi raggûj (télécommandées) melmi ran . 2002 Tawiza. .
