 uïïun 172, tamyur 2961 (Août 2011) Tamazight tamssizwarut: asmkl asfãaïi n tvawsa n tnttit di tmnäawt tamaynut (Le traitement sophistique de la question identitaire dans la nouvelle constitution) yura t: mupmmd budhan ula d yan ur itsba (nier) is tamnäawt tamaynut tfka i tmazivt yat n tasartut (place, statut) d tamdagt (positif) aïïas, mqqar tisiwav (obstacles, entraves) n usaäuf (loi) i tgga dg ubrid n usnãb (officialisation) n tmazivt. maca imala (mc) asnãb n tmazivt - s urda as itwsns -ur itnfa (être utile) tmazivt, xf uswir n tnttit. acku awank ad isul d aorab. s unalkam, tmazivt ad tili d tunãibt di yan n uwank d aorab. addad amatu n tnttit n lmrruk ur itssn, dix, ula d yan n usnfl. u akd waddad ad n tnttit taorabt n uwank amrruki, ad tsul tmukrist tanttyant (identitaire) i tuva iqqn ad tt tfra tmnäawt tamaynut. arni xf manyad is ntaf tamnäawt tamaynut ur tsnis (appliquer) ula d min d yusin dg inaw agldan n 9 krayur mani issiwl xf tmazivt zund aoqqa n tnttit tamrrukit. dg ukud i tuva ntggan ad tssnãb tmnäawt tamaynut tmazivt d tanttit tamdadt (véritable) n lmrruk, tçwy (surprendre) anv tmnäawt ad umi tssiwl xf tmazivt am yat n tisgrt (composante) waha ttas d oad awarni tioourba d lislam, s unmgal n tidtt tamazrayt (historique). ttini tmnäawt tamaynut is «tanttit tanamurt tmun s ufsay n qao tisgritin taorabt - tislamit, tamazivt, taspëawit tapssanit, u tbvr (être riche), s imuvln (affluents)» nns tafriqit, tandalusit, tudayt, tagrakalt (Méditerranéen)». tad d yat n tigzi (Compréhension) d tamikanikit i tnttit, txççar var s am tmarniwin tuïïunyanint (numérique) s yan n wudm ur dig s bu tmççult (logique). man tmççult i ysszrayn (au sens de "justifier") asiwl xf kraä waha n tisgaratin u d imuvln u maci kuç niv sa (sept) niv mraw...? oawd, mamc nzmmar ad nssiwl xf amuvl afriqi u nkni ntddar di ifriqya u ntta (appartenir) vur s? lmrruk d azyn zi tafriqt. dix, ur tzmmar ad tili tafriqt ad d yan n n umvl n tnttit tamrrukit. acku tafriqt ad nttat s imant nns d tanttit n lmrruk. maca asmuqql n tfriqt d amuvl iskan d is lmrruk ur intti vur tfriqt ad, intti vur ca n umnçaw (continent) nniän, amnçaw man tnnant timizar taorabin. acku lmrruk d aorab di tnttit nns, ntta ur intti dix vur ifriqya acku aorabn ur nttin vur umnçaw ad. tamanäawt tamaynut tkks tawinst "lmvrib aorab", maca awarnas tnna is lmrruk ad issmvar assavn nns n tnttit i t itqnen vur tuoourba. qao manayad iskan d is tamnäawt tra ad anv tini is lmrruk ntta lbd isul taorab. lislam tgg itt tmnäawt tamayntu d ntta d tanttit tamzwarut n lmrruk. maca tasartut ad i tfka tmnäawt tamaynut i lislam itarra t d yan n ijjd (religion) d amrruki waha u maci qao n imslmn dg umaäal, acku tanttit nttat ttqqn vur tulmist (spécificité). is dix lislam d yat n tulmist d tamrrulit waha? d tad d tanmgalt mani itawi usmuqql n lislam d tanttit n imrrukiyn. amntil (cause) n qao tinmgalin ad n tmnäawt tamaynut is itwari s yan n wudm d asfãaïi, mani iticc u itkks min iticc, issntam u itsbba min issntam, issnãab tmazivt maci itgg as tisiwv vad ysbddan asnãb nns, tissikiz s tnttit tamazivt maca puma ad tt iwddar jar waïïas n imuvlan d tigritin. agmuä (résultat, conséquence) is tamanäawt tamaynut ur tfra (résoudre, solutionner) tamukrist n tnttit tamrrukit. 2002 Tawiza. .
