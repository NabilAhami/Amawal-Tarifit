 uïïun 155, krayur 2960 (Mars 2010) Tamazight Tanfust: Nec d umalway Yura t: Yusef Oazz Eddin*, Asughel: Said BELGHARBI Tanfust a am marra tinfas nnidven, amalway (leader) ittwassen di marra timewwa n teghremt (tamdint), mani yeddar am marra imezdagh, zeg wami inghigh ijj n utarras deg ubrid, war t ssinegh min yeona, zi lltva nni d nnec twaligh abandu n umalway yuli di temdint a, idwel ittwassen am yetran n tmedditin ghar gguren imsebriden zi mxumblen isvurifen. War ssinegh mayemmi war nnigh deg akud nni i marra lghaci nni igg itexsen amalway: -“D nnecc i yenghin midden nni, suyenni d nnec ig itxessva ad yili d amalway nwem, war yelli d wa!!”. Maca war ugzugh i mayemmi deghya dewlegh ura d nnec am ismghan nnes, kessigh as di txuncay, am marra imezdagh!!. Ttugh war nnigh deg usenti n tenfust a, nec d umalway nemsarwas am wakniwen, ntawi d di tefraz d tmijja n aya-wya nnegh… tugha tnnujiwegh ghar marra igrawen nnes; nebla ma ad issen hedd, d nec i tugha yeqqaren marra tinfar d ilaghan nnes i tugha tarigh s ufuss inu!... Amalway d aqubbani deg iyyar n tinneflest (magistratura), axmi nni ghar s d ittemcetca weghref (cceob; peuple) nigh d ayt n imenghan u d yenni yennughen deg imxumbal n tudart nsen… taffed amalway war yessin manis i ghar issenta ad issefsi timukrisin a, icemmez deg imeswan nnes (moustaches), isniomir axmi yessen, maca war yeqqar walu, itarra xaf sen: (War yelli d wemnus, war dinni twiligh min xef ghar temxeyyaqem, assefsi n umxumbel nwem ghars uffugh!...). Ikk d wezlay n imalas d min arbun n wussan, idwel umalway d amghar u d akbab (symbole) n teghremt, marra tarran as taynit, tettaren as tudart s uzir d yidv, axmi ikides tilin; teggen ixf nsen texsent s wattvas; ad tawalid tittawin nsen tareqriqent s imettawen n wecmat. Amalway war ihemmel ccum n lbarudv d uqartvas d wexfif, axmi nni yessen ira dinni imenghan issewjad ad imnegh s isekkiyen (épée), mammec i d tugha tmenghan imgranen n zic n umezruy, s uyenni igg yettvef di marra min yellan d amenzu, u d suyenni igg itwala marra imenghan am ict n turaret, war ghar sent bu uttvunen (s), war yessin amezwaru ura d ameggaru nsent!!... Icenga ugzun i tmukrisin n umalway di turaret n imenghan, suyenni ikides tmenghan s taghara (Manière) nnes deg umcubbec, dewlen ccaten s umrig (des armes) am wenni i zi itmengha netta. Trigh ad iriwen ussan n imenghan n umalway nnegh iked icenga, imenghan itekken ddaren abarru n wussan s iziren d tmedditin, mani ittmarni ughewwegh n lghaci, d umsawal n isekkiyen axmi nni itqarqub uzzal nsen iked aya-wya deg wadvu n ujenna, d ilaghan n tmijjwin nnegh itettaren s tudart i umalway nnegh, imgranen (guerrier) s tadusi nsen sghuyyun axmi nni twalan idammen nsen tazlen, tmudjughen; starturen xef ucar n tmurta, ad teslid tcemcimen s ca n yiwalen war d asen itgzu hedd!!. Tugha ntegg mammec i tugha llan imenghan n zic, iteffegh d ca n umgara zeg ugraw n imerdas, iqqar s taryazt d tghuyyitin nnes: -Wi izemmarn ad kidi immughzel, ad yeffegh ghar ammas n ubaraz?!. Ammu id tugha nssenta amcubbec nnegh… Abarru n imenghan i negga deg wattvas n idughar d imucan, maca war ssinegh amalway nnegh mayemmi tugha itexs attvas n imenghan igg ntegg deg ugensu n tnezruft (sahara), war zemmaregh ad ttugh ussan nni di tugha ntghima dinni nweddar i wattvas n wussan, deg usezghel d tsammart n tfuyin, ar gha mmten iyysan nnegh s ujeyyef n ufadi d ulazvi?.. Radas inu i tudart n umalway yejja yi ad ssnegh abarru n tmukrisin d timeqranin xef umeddur nnes iffaren, zzay s ura d min yeqqnen ghar tudart n taqquyt nnes (sa vie sexuelle), ssnegh ira tugha war izemmar i ca!. Di marra agensu n tqesfin n umadval, ixessa igelliden ad ghar sen tili tadusi d tizemmar am ikidaren, ad zemmaren ad zzay s mercen abarru n tywin d timgharin deg ict n tmeddit, maca amalway odvarent tizemmar nnes!. Amtawi (Accord) yeqqnen jar ayid akid s; netta ad iffegh ghar texxamt nnes n yidves; axmi nni d ghar yendver yidv n tmeddit, mahend ad issemghar zi taryazt nnes axmi nni ghar yettves iked ca n teslit zi wattvas n teslatin nnes. Lltva di tilin marra imezdagh trajan uffugh nnes u netta yeksi jar ifassen nnes takettant, tbeqqedv s idammen n tuozariyya nsent… Maca di tidet, wami nemsarwas nec d netta di tefraz nnegh, d nnec i tugha akid sent i tettsven, itettar zzay i ad iffegh di test n texxamt, tugha textutturegh am uzrem ghar rectu n yidves nnes, maca wami tugha tettsvegh iked tislatin d tayywin nnes; tugha tacigh akid sent trunt s tiqqest yarghin d alsawen n tmessi deg wul nsent, tesligh deg unggaf nsent acetci itemsawal s imettvawen jar tmughrin n tin nsent, amettva nni immexsen remrac n ughir!?… Deg umeddur n umalway teffar xafi ict n tmeslayt, d taxxamt d yussin zzat i mani igg itettvesv, taxxamt a marra ussan nettat teqqen, war yeksi ura d ijj asarraw n tewwart nnes, war yessin ura d ijj min day s i tegg, yarezzm it axmmi yessen war dinni lligh di teqseft, nigh ghar s aghmis aqqa yi iggwjegh xaf s, ura d nnec teddugh d imewddar deg imunas n tudart inu… Iked uzlay n wussan, d wattvas n umyawaz n imenghan i ggigh iked umalway, nec i tugha war day i izemmar ura d ijj s tadusi inu ad day i yarna deg umughzver inu, nigh ad day i yeghdver ghar ucar n tmurt, maca deg ijj n lltva msagaregh iked ijj n umgara (guerrier), itsvudv am uyenduz n idurar, s ighen nnes izemmar ighedver ayi ghar tisi n tmurt, rajigh ad day i inegh s usekki nnes, uca yenna yi s ubusvar n wawal: -Upu!! War cek neqqegh ca, maca ad ac kksegh idvewdvan n ufuss nnec afusi!.. Ssutaregh ghar s tudart, d suyenni war day i yenghi ca!!. S tiqqest n umenghi nni; qqimegh abarru n wussan d nnec ttsvegh di tassut n umalway, war d arrigh taynit inu i marra min imsaren dinni, deg ict n tmeddit kkaregh d s tigwdi zeg uskuti inu, awarni wami urjigh ict n tarja, srigh day s i ticti n tsira n umalway; oeffsent xef tisi n teqseft, arezmegh tawwart, twaligh yudef ghar umadval nnes, ghar tiffar n texxamt nni, udffegh akid s ghar dinni… Zrigh attvas n tismaqlin, uylent xef igudar n texxamt, marra ksint timitar n umezruy n imenghan nnegh, dinni tiwalfin day sent nec d umalway, ict nettru day s am isegman, ict nesfarnin day s, tinnidven n cennu (Chantez) day sent, nigh n turar day sent am imezyanen igg itarran mezri i temzvi nsen. Jar tilwihin a marra, dinni ict d tamenzult, udsegh udem nnes; smuqlegh day s attvas; twaligh dinni ijj n umgara (Guerrier) tugha olayen ad day i ikkes idvewdvan n ufuss inu afusi, s ijj n usekki (épée) itissiq jar ufuss nnes!!. Di tma d yusin deg umeggaru n teghmart n texxamt, dinni uylen t tismaqlin n marra tislatin d tmettvutin igg yemrec umalway, dinni ict n tisit tuyel zzat i tewwart n wuffugh, smuqlegh day s mahend ad day s xzaregh tidet n wudem inu, maca zrigh dinni tinni n umalway, tifraz nnes warghent; tiddi nnes teqqudvedv; udem nnes yessu s icudvadv n uxeyyeq, yejja tmart d tacemlalt tiriw xef idmaren nnes; yegga am ca n tesmaqalt texnunnes deg waman yarewyen, war yenni ad lami day i yessen, tugha tacigh akid s yetru s tiqqest deg ugensu n wul nnes, sseqsigh t: - Mayemmi…? Ismuqqel d day i s uxeyyeq; rxezrat di war twiligh taqsuhi nnes am tenni n wapli. Yenna yid: - Attvas n twaratin d nec tnuffaregh di texxamt a, mani war day i itwili hedd, ssizdigegh day s ul inu s imettvawen!!... -I manis dekkin ixsan a? Wami yexzar deg icedvraq nni n yexsan yeksi di turutv n ufuss ness afusi, yarred yenna yi d: -Yina d ixsan n idvewdvan nnec!! -Maca war yelli mayemmi d asen ghar tarred taynit, min yellan itif (importance) ad tilid cek di tcuni!!.. -Maca!?... -War kessi s imunas, d cekk i ghar iqqimen d aselway n teqsefta.. S tumart d usfarnen, yarred xaf i: -A yamgara inu, yuccel xafk ad tessned aqqa rexxu nezdegh nec d cek deg ict n teqseft tussed deg ammas n tnezruft, tissraw s razagh n tzawa, ffghent marra imezdagh. -Maca!!. -Marra min n teddar ammu; tugha yella d tarja n ijj n wass, nigh d turaret i nessenta zi temzi nnegh oad tuggi ad tenfarar, nec eggigh day s ixf inu d amalway; cek d amgara; d izem inu… -War ugzugh ca i wawal nnec, maca mayemmi twaksen ammu idvewdvan inu? -Wami tugha nella d imezyanen, n tirar nec d ccek di teqseft a, deg ict n twarat tudvnedv s ijj n rattan d amizi (danger); war ghar s bu ugenfi ura d asafar, yejja idvewdvan nnec qettvwen zeg ufuss nnec, wettan ijj awarni unnidven; am tefray n tseklutin. -Maca!!... I mayemmi oad aqqa negh neddar di teqsefta ar ass a!?… -Wami tugha negh d imezyanen iggwed ugellid n teghremt zi turaret nnegh d imenghan nnegh, iggwed zi tumayt n tmacaputin nnegh, ittghil imenghan nni ad effghen d ttidet deg imal n temdint, ad ilint d amizi xaf s axmi nni ghar nemghar, maca inder anegh di ugensu n teqseft a, waxxa ccetcan nigh run ghar s lwacun nnegh huma war danegh ittejji da, s umyawaz nsen i neqqim neddar ar ass a!!.. Yeggwej umalway xef rxezrat inu, ssentigh twaligh mghar tifraz inu di tisit nni!!. Ffeghegh zi texxamt nni, undvegh xef tmewwa n teqseft, sijjegh zi tburjet ghar lagwaj n tnezruft, mani yessu yejdi ijartar n usezghel nnes; yulmen s tesfifin n izenzaren n tfuyt, smuqqlegh dinni marra imenghan nnegh izelyen, am ict n tidet teddar jar tin inu, twaligh dinni inighiten (Les victimes) n imenghan, tesligh i tghuyyitin n yenni yennuryen di lwacunat nsen, tesligh i ushinhen n yeksan trabaoen s ticedvyin nsen deg ioerbab n toejjuja n yejdi… Deg ammas n manyenni marra, twaligh ijj n tunt (des espaces) war ghar s bu timewwa ura d iymaren, itzemma s tadusvi nnes iqeshen am wuffar aswingem inu dinni ghar s yarewren, deg umeggaru n wesvwadv inu twaligh ict n tseyynut n yejdi; tuddumed am wenzar, ghelyegh deg wul n tharyatv nni, am wattvas n tuddimin n yejdi ssunt dinni ixf nsent di twangimt inu, zedghent deg ict n tumayt n udghar ugha war llint tefraz am tidet. * D amyaru d AKURDI. 2002 Tawiza. .
