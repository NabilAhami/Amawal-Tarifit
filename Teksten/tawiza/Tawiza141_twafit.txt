Yura t: Said BELGHARBI Af it... jar idvewdvan n ucennay d tazvuri turi d, af it d timitar jar tefray tzura tecci t, af it d tawalt iles n wass itefz it, af it d d tidart deg iyyar d tufrint, af it d tighit i weydud d tasertit, af it d isefra zi tcetcan initcan x imal n ijjen d wenneghnit, af it d iseqsan n teyywin ghar war tedji temwarit... Af it di tmaccaput n imsebriden, ijjen yenna t d arubi, ijjen ijji t d twafit, af it d areqriqen n waman afadi yeswi t, af it d ileqwaz n weghrum lazv ijjiwn it. Af it d anufser n temzvi di tewsar ttissiq d titrit, af it d mezri zi tefras inu x tefras nwem yefruri d, afi t d askil n uraghi di tayyut i turid... Af it d aqessis ibettva deg ibriden, i cem yenna m dvfar tansa n tebritin, i nnec yenna yi zvwa tennednit. Af it d anessis ghar yimi n ugelmim itmudv deg waman, itari zzay sen deg ucar timaccaputin, itari i yenni yeghrvin min war ghrvin, af it d asemmidv izeppar am usegmi deg wezgher n tehsinin. Af it tedja axmi nni war xaf s tarezzud day i nigh day m, tteffar, trakwar, axmi tteca d areqqas ghar s tusi d, s uzawar zi ccurent tebratin nnem tisid. Af it d tumart x idurar n Tamezgha ttessu tifeswin, af it d axeyyeq deg izerman n wassam tesrugha tifawin, af it d anuhmez n weghyay deg ucar issutar tufughin. Af it d lgha jar wadjunen n umedyaz s zrayar yedz it, af it d anggi n imunas asif yiwy it, af it d afran s inuda n wazal izewy it. Af it d rxezrat jar ayl d ughedjuy n tfuct d afulki zi tessenta tmeddit, af it d tliwliwt di tfugla (Cérémonie) n tmeghra tarzef d d taslit, af it d asezghel (Chaleur) i wul ijarsit, af it d tazyudvi di tsegnit n tzizwit, af it d asfarnen izedghen x tmughri n tsegmit, af it jar tiyyita n wenzvar teccat d tuddimit, af it d tilelli i tudart n unecruf izzuyt it. Af it di min yedjan qao tedja d tighennanin, af it mara war ttufid arbu hnuc akidi ad tnaf ikd twafitin.
