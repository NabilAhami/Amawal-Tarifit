 uïïun 125, tzayur 2957 ( 2007) Tamazight tamssizwarut timanit i lmuvrib niv i tmnaäin n lmuvrib? (Autonomie du Maroc ou autonomie des régions du Maroc?) yura t: mupmmd budhan iïïus (iggut) uskasi (débat) xf timanit tamnäayt (autonomie régionale) i tmurt n arrif. tangmit (revendication) n timanit n tmnaät n arrif ur tlli d tamaynut. tssnta zzat ujar mraw (dix) n isggusa. maca awarni tafaska (fête) n ilmahaln (travailleurs) tudf yan n umcwar (étape) d amaynu. ula d ayt sus ssntan tettarn timanit i tmnaät nsn. ayad ur igrç (plaire) ca i inbbaän aorabn. suynni ssntan ttinin is ur izmmar ad tili timanit tamnäayt di arrif acku ntta d yay n tmnaät itddarn s lkif (kif) waha. oawd arnin ttinin is tangmit ad tra ad tbäa lmuvrib imunen. maca ula d amussu amaziv itettarn timanit tamnäayt n arrif, issgdil (débattre) tivawsiwin n tämsa (économie) iqqnen var timanit tamnäayt u ijja tutlayt d tnttit (identité) tamazivt i tuva awarni tlalit n tngmit n timanit n arrif. man inlkamn (conséquences) vad yilin var timanit n arrif xf tutlayt n tnttit timazivin? asmi vad yili arrif d tmnaäin nniän s timanin nsnt, mamc vad tili tnttit n tnbaät tanamast (pouvoir central)? ad tili d taorabt. dix, angraw (système) n timanitin ur issufuv tanttit taorabt zi lmuvrib. s umngal (au contraire) itarra tt tdws (forte) acku ttavul d tanttit n tnbaät tanamast. arni xf manaya abïïu n tmazivt var kraä n tutlayin di arrif, d sus d laïlaã. imazivn tavuln d yat n tadrsi (minorité) ttfk asn tnbaät tanamast taorabt izrfan nsn di tmnaäin nsn. max (maymmi) imvnasn imazivn itettarn tamanit ur llhmn (se rendre compte) i inlkamn n timanit i ur tnafan (servir l'intérêt) tutlayt d tnttit timazivin? acku isin amdya (exemple, modèle) n sbbanya (Espagne) d apttu (modèle à suivre) i timanit di lmuvrib. maca ttun is tlla yat n tmçla (différence) d taxatart jar lmuvrib d sbbanya ur nzmmar ad kids nsns (appliquer) amdya n sbbanya di lmuvrib. max? di sbbanya tutlayin d tnttitin n tmnaäin n kaïalunya d lbask ur llint d tinamurin (nationales) vmsnt (couvrir) qao tammurt n sbbanya. iguttutn (limites) nsnt tbddan var tmnaäin n kaïalunya d lbask waha. dix, angraw n timanin di sbbanya itnafa tutlayin d tnttitin n kaïalunya d lbask i tuva angraw n tnbaät tanamast igdl (exclure) itn u ur izzay snt issikiz (reconnaitre). uma di lmuvrib, tanttit tamazivt ur tlli vas (mvar) di arrif nvd di sus. nttat d tanttit n lmuvrib marra, n imuvrabiyn marra. imala nsns tamanit di arrif d sus ntarzza u nbïïa tmazivt u nsslkak (affaiblir) it, u nssdwas (renforcer) tanttit taorabt n tnbaät tanamast. iqqn amzwaru ad nsns timanit i lmuvrib s uzarug (indépendance) nns zi tnttit taorabt i iddaffar. asmi vad ivul lmuvrib d yan n uwank (état) d amaziv s tnbaät tanamast d tamazivt, oad i vad nmmutti var timanitin n tmnaäin acku tutlayin d tnttitin nsnt ad ilint d timazivin am qao lmuvrib d imuvrabiyn. 2002 Tawiza. .
