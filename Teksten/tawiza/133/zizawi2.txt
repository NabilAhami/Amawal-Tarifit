 uïïun 133, smyur 2958 (Mai 2008) Tamazight Asfidjet Yura t: Oebdlmuttvalib ZIZAWI (tasdawit n Ujda) Asfidjet d ict n tesvmudvt n tenfas tiqudvadvin n umyaru Saoid BELGHERBI, Tinfas a tghimant d asfidjet ghar idvennadv d wass a, d asfidjet ghar tiwecca. Awarni "Aswadv yebbuyebhen" irur d udlis a d tarwa d tamaynut deg iyyar n tsekla tamazight ittwarun. Tinfas a jar ifassen nnegh attvas zzay sent twarunt ajemmadv in i waman. S manaya ntaf day sent qasu n uzwag d imxumbar ghar s iqqnen, ntaf day sent oawed amarg n umezwag(1) ghar tmurt nnes| Adlis a day s tifras n redcur n Arrif iweddvaren deg ixfawen nsen, am uweddvar n imezdagh nsen di timura n midden. Saoid BELGHERBI di tira nnes itayem d cwayt zi tenfas n zic, s manaya ntaca axmi aqqa negh zzat i tenfas tugha d anegh toawadent tihennatin nnegh. Ca n twaratin nnidven ntaf amyaru nnegh itari tinfas, maca s ughanim n uzrawi, u manaya yewca ijjen n udem icna i wesfidjet. Tinfas a ntaf itent umsent s wattvas n iseqsan iqqnen ghar tefgant, ghar Arrif jar idvennadv d wass a, iseqsan n umezruy iweddvaren, iseqsan n tamust itwagzen(2). Tinfas a d tiseghnest ighensen x idmaren n tsekla tamazight ittwarun, d tabrat n tayri zeg wur yarghin s ucemmudv n tira, d ict n tigi n uzbu(3) i tudart n tmazight. Saoid BELGHERBI yexs ad yini deg udlis nnes: Cem d tfuct inu, d taziri inu, cem d tudart inu, d raxart inu, cem d yemma inu a tamurt inu, ad teqqimed i lebda d tiggezt izedghen deg wur inu a tamazight1inu. Notes: 1- Amarg n umezwag : la nostalgie de l’émigré. 2-tamust itwagzen : l’identité tatouée. 3-tigi n uzbu : acte de résistance. 2002 Tawiza. .
