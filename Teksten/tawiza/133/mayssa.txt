 uïïun 133, smyur 2958 (Mai 2008) Tamazight Temrer n tucemretc Tura t: Mayssa rachida EL marraki Wi t inghin? wi t ijeyfen? maca netta yemmut, yegga mlih imri war yemmut iri d attvas xaf s!! wi kid s yengha? ijeyyef ixf nnes, dipa… senj i wurtu nni swizdven d ghar s waghrawen n dcar, pnunyen d imettvawen aked yeghzvar, xesyent tfawin n buhber Saoid ma nghin t nigh ingha ixf nnes???? Wi ad kid s yemmuten idwer ifaq d di tirja nneghni d ticemrarin tbarcinent, maca ass a sarewten zzubeo hencen anedrar, ghzin tisarfin ghzin timedvrin nneghni wami Saoid ingha ixf nnes, netta i ca itarezzvan tisudvar itghedjar tazeggûrt ifares idvewdvan nnes zeg fassen ighiswen, mechar ifardv di tazdugi n temrer, mechar ighars di reshun n tidet iggin iqetvwas, iccat izra s tfednin isendaf i wezvru iyezzem tireghwin. Wi yjeyfen Saoid, mant ajeyyef zeg jeyyef?? tghufi war tneq maca ca n twaratin teseddar. Takemmict n iseqsan teqed day sen tmessi, ad texsi mara salant tmenna tugha yeqqar rami yemmut, yuyer ixf nnesd ayar urtu itru x tefray nnes, Saoid tezm it tudart tzawaren t wawaren nnes ijjen itbarkikidv di tmijja ijjen ihrudjed d x yires ca nzerqen deg metni yeffuden tara n wayam tacemratc. Mcubcent a s tefras x wudem mant udem izi ijeyyef, mant tudart izi yemmut?? isghery as tarzugi nni id as iscerwen anixses teseghri yas anexrio x tmewwa n wur. Saoid ijeyyef artax ca izkwyen anedbu di tendutin n usitem, isecmedv ifassen isqarquben tiwwura n rirart tacemratc. Saoid immut tlulluct n raxart, tlulluct tisraw ixsi wur nni yarghin tfawt n webrid ghar tewwart n reqwar isexsi day s turut nni n ulellec acemrar, teghba tsarrawt x yiri, yeghba uxarresv deg xarresv mxumbrent txarrisvin, irur d usiymi iffegh d zi raxart izregh d s tefras n umsarwes d ca zeg udem n tvya itawi ghar tucemretc, axmi temsafadv i wghedjuy x tidet min di mma d raxart mamec mma, axmi tmuttuy zi raxart ghar raxart d tcuni tessu tucemretc…… (Tanita temecsid zeg ict n tidet n tensfust temsar deg ayt teyeb netta ijeyyef ixf nnes, tanfust nnes attvas i day s n ineghmisen u uran xaf s yeghmisen n ughezdis a zi arrif.) Mayssa rachida EL marraki, 16/03/07 2002 Tawiza. .
