 uïïun 173, tzayur 2961 ( 2011) Tamazight «Afdudr n tmzvla», timdyazin n ben zroual Yura t: Mohmed arejdal ar itbughlu igr n usnflul amazigh s tlalit n tawckint n tmdyazin tamzwarut n mass abdellah ben zeroual lli mu igga s wassagh: «afdudr n tmzla» lli d idvi gh ugadir asggwas n 2010 . Amdyaz abdellah ar izzuzzur tujjutin n tguriwin n mdyazt gh igr n tirra d tsakla, ar igrra amud n tgnsa d usnflul amazigh ligh tfdudr tmzvla gh ull ns tg anzgum d uswigm yusin tugt n wammakn, tg izlan nginin tyaranin s tutlayt tamazight isfawn izdgn lli d iswan gh igbulla n lattvlas amjapdi d suss; fad ad yziy watig n tutlayt tamazight gh ull ns, yasi tasafut n tmagit ns gh ufus ns. Tawckint tamaynut: (afdudr n tmzlla) tga ifghddi n umyaz ben zroual gh wanaw n tmdyazt lli d iffghn gh ungaz n tillas n tilawt. 2002 Tawiza. .
