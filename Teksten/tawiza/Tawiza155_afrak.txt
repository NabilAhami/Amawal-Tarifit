 uïïun 155, krayur 2960 (Mars 2010) Tamazight Tmsmunt afrak (Association «AFRAK», membre fondateur du «collectif Azaykou des associations amazighes) Anqqis n tsutlt n usmuttg f tutlayt d usslmd n tmazight Mklli t tsml g ulghu lli tsizzl i ugdud n masst ula win brrva, tskr tmsmunt afrak yat tsutlt n usmuttg f tutlayt d usslmd n tmazight g uzddugh nns gh masst ass n 04 d 05 brvayr 2010, ittuzvlayn i islmadn d imzrawn d willi tt yiwin g tutlayt tamazight. Macc idrus ma ihadvrn g islmad (ur nssin max) slawan akkw willi isaqran d willi ra isaqqra tamazight g usslmd amnzu, mqqar ar bdda tsmummuy tugt gisn f imukrisn lli darsn illan g usslmd d ussghr n tmazight (tifinagh, didaktik, tajrvrvumt, amawal…). Mqqar d ghikann, imadrutn lli yamun g unmuqqar ad n usmuttg, slawan akkw imzrawn n usaka n tghuriwin timazighin n tsdawit n ugadir, iwin nn tayafut mqqurn, fk tayafut ula nttnin, sattuyn aswir n umyanna d tussna f tutlayt tamazight, acku nttnin ad igan imal n usslmd d ubughlu n tmazight g kraygat asays d tmssdusin n tawunkt tamghribit (tinml, asmal, taghdmt, taghmsa, …), mqqar asn ur ttufka tawit g tmldiwin n uswwuri d unnurzvm n tflwin n twwuri. Tasutlt ad, lli f cayfn islmadn Hassan Burkoa, obdsslam Bumisvr d Lorbi Mumuc, tbdva f mnnaw igumn (axes) zund agwmmay nghd askkil n tfinagh, asnmarra (orthographe), tajrvrvumt, tasnayt (tasddast/syntaxe) d tdidaktikt n usslmd n tmazight. Dat ad ssntint tsawurin, tlla zwar, g tifawt n wass amzwaru, yan usurs f waddad n usslmd n tmazight d imukrisn nns. Mdiwln gis imadrutn d isalmadn f mnnaw n imukrisn d id bayttvar n tmazight g tinml n lmghrib, tanamurt ula tusligt: amukris n ussinf n usittvn n tutlayt tamazight g lbiltan, asmmaql andram-adamsan d win isvahatn lli illan dar kran id babatns d imarawn n inalmadn, azzmzviy n watig n tmazight, asyafa n thggart n ixfwan nnsn, amukris adlsan, asrtan d asikulujiy lli illan dar imghrabiyn slwan akkw islmadn n tmazight lli izzmzviyn atig d tawwuri n usslmd n tmazight lli fllasn illan ad tt skrn, ar rzvzvan tayri qqurvn d tirit mqqurvn lli s ran inlmadn ad aqqran tamazight, tamasayt n tmawast n usgmi d usmuttg d tmssdusin lli as iwalan, ig amdya iggwran aylli mi itthiyyal Usqqamu unnig n usslmd. Amdiwl ad d umyanna iffagh s yan mnnaw isumurn zund: ad usizvliy (taxassvusv) dar uslmad n usslmd amnzu, ad tskr tmssdust tamasayt (nniyaba d tmawast) ismuttgn ddutnin, ad ismuttg uslmad ixf nns s ixf nns, ad mmaghnt tmsmunin, idvfurnt tawwuri n usslmd n tmazight, awsnt ula nttnti, ad tettufka twwuri n usslmd n tmazight i willi tt sawalnin d willi gis skrnin asmuttg slawan akkw imzrawn lli d iffaghn g isakatn n tughuriwin n tmazight g tsdawiyin n lmghrib. (Anqqis: Lorbi Mumuc, Tamsmunt Afrak Masst) 2002 Tawiza. .
