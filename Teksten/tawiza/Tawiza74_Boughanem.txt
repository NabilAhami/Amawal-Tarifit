 Uttvun 74, Sedyûr 2003 (Juin 2003) Tamazight Imazighen d tugherma n ileghman Yura t: Bughanem Xalid (Lefnideq, Tittvawin) Tasimyutikt d yat n tmessunt (science) i d yefghen ghar tilawt x umezgaru huma ad tegg yat n tighri d tamesniwt (scientifique) i tenfas (contes populaire).. Ztat' ma ad yiriw igar n temsenta ar wamud (exploitation) n wattvas n tsenfulin (expressions) nnidven. Amki (ammu), tesmerv anegh iferdvas (elements) ig isekkan (construire) tiskiwt (structure) n tenfas.. Tesmerv anegh assagh (relation) ig itsmunan imyagen (acteurs) jar asen deg wgensu n ict n tenfust, nigh n wengal, nigh n usaru (film)... Maca aseqsi axatar ig icardven (imposer) ixef nnes: is tzemmar tasimyutikt s mayn ghar s d tarrayin (méthodes) ad tegg asnas (aplication) i tidmiwin (penseés) nnes deg igar n umezruy??? Amezruy n imazighen s umedya (par exemple) zeg akud n wadaf n uumesdurrey aorab ghar tmura n timuzgha itghima d ij n igar d amiriw i d isfrurin kigan n tenfas n tilawt. Tinfas yeddar it'ent umazigh.. Amezruy n umazigh d tidett yedwer zund tanfust, awd yan ghar s tawegit ad yini belli amezruy nnegh war d itiwi deg tenfust. G.Camps, deg wdellis nnes "imazighen x tmuwa n umezruy" d yan n udmawen n tidmi ig issentem (affirmer) taghawsa yad s ij n useqsi, yuss d deg ammak nnes: mamec temsar ar ami yetwakkes wezwil (nom) n umazigh zeg wmezruy s wadaf n 10000 n waoraben, tugherma oemmars war xa s slen???!, d ict n tenfust d tajabalt (etrange). Mamec i d ayi ggin nek ig isulen ssawalegh tmazight aked yemma, aked ayt ma... tirja inu tarjigh zzag s, ssaghegh teznuzigh zzag s, ar ami arvin ayi ssawalegh x ixef zund yan irur gi Ssaoudi nigh gi Lyaman...? Nek war gigh aorab, maca waxxa yammenni, mermi ma ssiwlegh ssawalegh s wassagh nsen. Mamec ig ggin ar ami arvin ayi ttinnigh ztat' umadval marrva: "NEK D AORAB" s iles inu yeffaryedv. Tesfariydv it' tutlayt nsen netnin.. tutlayt n inen id yusin -amuk id anegh nnnan deg wdellisen nsen - wwin d akid sen i "ayt barbar!!" tugherma nsen!!, d ileghman nsen, d dahis nsen... Agmudv (résultat): imelyunen n imazighen fesyen deg tugherma!! n 10000 n ayt n ileghman. D tidett, ur yedji marrva imazighen deg tidmi n tuoerba.. Maca inen ifesyen, is zemmaren deg ass a ad dvurven ghar uwanek (état) nsen n tidett??!! Adfel waxxa yefsi, yedwel d aman, zemren waman nni ad fuwaren ar ugennna huma ad pwan i tikkelt nnnidven d adfel!. (Bughanem, ass n 8-5-003) 2002 Tawiza. .
