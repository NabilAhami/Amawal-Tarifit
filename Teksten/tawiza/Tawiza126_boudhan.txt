 uïïun 126, Mrayur 2957 (Octobre 2007) Tamazight tamssizwarut: aslmd n tmazivt niv aftrul n talssint wis sin! (L'enseignement de Tamazight ou le pétrole de Talssint bis) yura t: mupmmd budhan dg unbdu n 2000, ddarn imuvrabiyn tavilt (illusion) n twafit (au sens de "découverte") n yan n wasif d amqran n uftrul (pétrole) di tmnaät (région) n talssint. agllid mupmmd wis säis issiwl s usulf (occasion) n tgrut (tawafit) ad inna is "aftrul n talssint iggut (en grande quantité)". maca tumrt (joie) n imuvrabiyn ur tvlel (durer) ca aïïas. awarni sin n iyurn waha, yuman d is aftrul n talssint d yan n usrks axatar ujar qao dg umzruy n lmuvrib. aftrul n talssint tuva t d asudds (escroquerie) d amqran. ammu i yuval uftrul n talssint d yat n tarja tcna (tiçil) tkka taodu. max nssiwl xf uftrul n tlssint dg izmz (temps) mani asntl (sujet) nnv d aslmd n tmazivt? acku aslmd n tmazivt, i isntan di tzayur () 2003, imsar as zund min imsarn i uftrul n talssint nican. aslmd ad ivna (créer) var imazivn ict n tadala (anaruz = espoir) tmvar aïïas, anct n tadala i tuva d ivna uftrul n talssint di 2000. Maca, am imsar i uftrul n talssint, aslmd n tmazivt idwl, awarni sin n isggusa, d asudds u d asrks. tuva t d tivlt waha. Max? asmi issnta uslmd n tmazivt di tzayur 2003, ismuttg (élaborer) uvlif (ministère) n ulmud yan n uvawas (plan) ibdd (basé) xf asfsni (progression) d usmtu (généralisation) n uslmd n tmazivt zg usååas var wnniän s ij n wudm ad kids ivms (couvrir) uslmd n tmazivt qao almud (enseignement) amnzu (primaire) di 2009 - 2010. ittini oawd uvawas n uvlif n ulmud is ad issnta uslmd n tmazivt dg uswir (niveau) almmas (secondaire) di 2005 - 2006. ayad icna u içil. akd uvawas i ismuttg uvlif n ulmud, aslmd n tmazivt ad yudf aså^åas a (2007 - 2008) aswir wis smmuã (cinq). maca iqqn ad nssn is ommars (ursar) aslmd n tmazivt ur yzry (iodu) aswir wis sin. dix, ur izmmar ad in yili uswir wis smmuã asl (sans = nnhra) ad in yili uswir wis kraä. ayad ira ad ini is aslmd n tmazivt ibdd var uswir wis sin. uma aswir almmas, iqqim d tarja waha. Ammu, i d ittffu, dix, aslmd n tmazivt, am uftrul n talssint, d ij n usrks u d ij n usudds d amqran. ur iqqim attayn (presque) walu zg uslmd n tmazivt, vas isrkisn (mensonges) d tivalin (illusions). max ssbrirn (faire échouer) inbbaän iorubiyn asnfar (projet) n uslmd n tmazivt? tiba (absence) n waqqas (volonté) tasrtit (politique) d nttat d amntil (cause) n ubrir (échec) n tskkirt (opération) n uslmd n tmazivt. tilawt (existence) n waqqas asrti d tafada (condition) tmzwarut i wsmurs (réussite) n uslmd n tmazivt. asl waqqas ad, itvima uslmd n tmazivt d tirart tssufurhut (occuper par un jouet) zzays tnbaät taorubit imazivn. puma ad in yili ca n waqqas asrti s tidtt n uslmd n tmazivt, iqqn tanbaät s imant nns ad tili d tamazivt. dix, yuccl (foir) i umussu amaziv ad ivns afad ad tili tanbaät d tamazivt di tmurt tamazivt n uvrf (peuple) amaziv. riv ad iniv is asmzv n tnbaät ntta d tafada n usmurs n usnfar n uslmd n tmazivt. 2002 Tawiza. .
