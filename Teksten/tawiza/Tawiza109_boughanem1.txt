 uïïun 109, smyur 2006 (Mai 2006) Tamazight Aghmis n SSVABAH nigh aghmis n useghlel n tuoerba Yura t: Xalid Bughanem (Tittvawin) Ur iqqim walu yennuffar. Afrak n imazighen s tmeggit nsen iggur itmarni.Tanbadvt, ikabaren, timesmunin d aydud ammen imun.. marra snen aqa ur din tedji timedwelt ghar deffar. assnnatv oad tugha amazigh ighettar ixef nnes, itmehray hma war d as d inejjem ci n wawar zeg wqemmum ad as yeffegh dag s tujjut n tmurt, dag s tujjut izvuran.. u netta aqet' ztat i lghaci. Tugha t' hma ad yesken tighri nnes x ayt mas yessawal mamec awal nnes war itiwedv ad yetwagza..yessawal "tahedvrit"! Itettvef aghnan aked ixef nnes. Itxes ad yiri mayn ur yedji . Ass a, drus n imazighen xmix temmedvren tezgawt yeqqim ur yessin mayn iteona, kkes inen i tesdargherv ci n tedyulujit d inen iteggen ixef nsen "war sedjegh, war zrigh". Maca waxxa yajenna d ajenna, tfuyt d tfuyt, tmazight s izvuran nnes i tedja, imazighen ammen nnedvni ..tawengit n ci yedvsen teggama akid s d tas tiktit: ajenna d ajenna, tfuyt d tfuyt, tmazight d tidet, d amezruy u d tagrawla mgal i yinen irezzun x ixef nsen barra i tmurt nnegh. Deg uttvun wis 106 zi TAWIZA, yura umat negh Mimun Amesbrid x usnebji i tegga ict n tesdawit nnegh i umegzar uttvun 1 n tmazight deg igar n tarezzut "tasnawit" mass Ali Fapmi xacim, d umsawal i kid s yegga Hasan Loettvafi zeg wghmis n Ssvabah. Umat negh Amesbrid amuk id anegh yesnum gi tira nnes s taorabt , s tmazight nigh s tefransist, timenna (dires) nnes tughatent marra deg wraq nsent.. ur din yedji umentil (cause) ad anegh arzven azedjif s inawen n tghuni n tesdawit aked temnadvt nnes (deg igar n tusna, usexnitar "economie"...) deg akud d ijj ad raghan i yejjen tirezzutin nnes d timenghiyin i temnadt, d timenghiyin i tgumi n temnadt. Nec urid xiyqegh mayemmi ict n tesdawit gi tmurt nnegh ad tragha i yuttvun 1 deg akkadv nnes i tmazight anect i xiyqegh maghar ad yekkar ij weghmis yewwedv ad yiri netta d amezwaru gi tmurt nnegh zeg mayn iteznuza s yedrimen n wattvas n imazighen, x webrid n tmesghit n uweghmis nigh x webrid uwawwaz (aide) n uwanek (etat) id as ticc, (ad yekkar) ad yewwet imazighen zeg worur !?? Izemmar ci n yejjen ad yini mayemmi Ssvabah mix twaggen tittvawin nepra ma ad nidar ighmisen nnidven ? Nec ugigh ad arreegh taynit i yighmisen ig yessawalen s yires n ikabaren. Ina nessen iten mani tessawadven. Ugigh waxxa ad arreegh tazwart x ighmisen yeddahhan ghar sen azarug x ikabaren d lmexzen d...maca tughaten aked ci n ukabar. Wen yennimen ag ci gi temzvi ad xas yeqqim gi tusar, nnan g awal nsen aoraben n zik. Deg attvas imuren, mara tufid ci n yejjen yeccat deg wkabar mani tughat', ssen aqa s tusmin x imenzayen (les pricipes) nnes ig yedjan alidv mgal imenzayen ig yetwarvem ad xasen yasem. Taynit inu amacnaw tenni n ssi Mimun tekka ar Hasan Loettvafi maci minzeg netta d ijjen zeg wneghmasen (journalistes) n Ssvabah ukan, ujar i manaya Loettvafi itegg ixef n uhennec (redacteur en chef ) deg wghmis. D tidet, izemmar aneghmas ij umur ad yari nigh ad yegg amsawar i tnafult n weghmis miked yedja asel (sans) a ghar s yirvi llawt x inelkamen (conséquences ) i gha yejj ar wattvas n iwdan.. maca mermi id tufu tghawsa n tira teqqen ar ijj ughezdis n tidmi ukan, s unegmedv (surtout) mara tidmi ya ur ghar s yedji raq di tilawt, tedvwerv d ij n wadvan itweddah (salir) zzag s bab nnes udem (image ) n uweghmis yemmi ig itari. Amuk id yusa deg awal n umat negh Mimun, Hasan Loettvafi oad war yewwidv ad yirvi deg d ij iyetri n ujenna n tughemsa tanamurt. Aghmis n Ssvabah x mixef yedja d ixef n uhennec, maolik ur yedji d Nini ad taf ur yewwidv mani yewwedv rexxu.Maca Nini itari x imxumbar n tmurt u s wawal n ayt bab n tmurt (ddarija) mermi id tawi, Loettvafi itamer (exploiter ) tnufsilt (occasion) n tawt (existance) n imegraden nnes hma ad yesseoda x anegh tadyulujit nnes ig yutan izvuran gi tuoerba. (nec ur djigh ci ag Nini mermi yessawar x tmazight, maca tidet ad tinigh: Ssvabah iteznuza t Nini ). Awarni i wemsawar ig yegga aked mass Ali Fipmi Xacim, yennurzvem as uoeddis i Loettvafi ujar ar tidmi n "tqawmit" taorabt, ismun rehnudv nnes yedda ar Masvar mani gha yemsagar aked uqawmi Usama Anwar Oukaca, akid s yegg amasawar ( Xzar amasawar i kid s yegga gi ssvabah deg uttvn 1840 -08/03/2006 ) d yusef Moatvi (xzar amsawar i kid s yegga gi ssvabah, uttvun 1846 - 15/03/2006 ). Iseqsiten i dasen yuca marra olayen x waddad (situation ) mindeg teddarent tmura n waoraben !! amaknaw imeghriyen n ssvabah ur ghar sen bu wemnus ghas amnus n tmunt n waoraben !!?? Maghar war yedda mass Loettvafi ar ci n dcar n lhusima ad din izvar isegman ruren d ar umadval deg ammas n uwedfel.. ad din izvar iwessura ig ibedden deg ij n uwakud deg udem n urumi deg wnwar, di dpar n ubarran, di buzineb d wattvas n iruqqa (ad ten izvar) tsummuten imunas nsen di barra awarni i 25 iyuren zi tarjajjayt n tmurt. Maghar war yedda ar ayt urir ad yegg ij n umsawal am wen itegg gi masvar ag lwacun n umettin ig yesduqquzen ighulad n ayt urir.. ghas ad ten yesseqsa, mamec temsar iqqim deg wbrid mani it' nghin ddebduben x as izan zi tufut ar ami tfuyt tedda ad teghli..! Nigh manaya netta qao war d as d yecqi . manaya war iteqqes deg ul!?
