 Uttvun 73, Semyûr 2003 (Mai 2003) Tamazight Tamessizwarut: Is d amezwar n uneggar n umyey Ufqir? (Est-ce le commencement de la fin du mythe Oufkir?) Yura t': Muhammed Budpan Amegrad a ur yelli d tamzaght (défense) n Ufqir. Netta d tamzaght n tidett. S unelkam (par conséquent) netta d tamazaght n tmazight acku d nettat yellan alidv (lebda = toujours) d taghtest (victime) n usentil (le fait d'occulter, de cacher) n tidette. Asulef (L’occasion) n wawal ad d may tefsar “Lahdat Lmaghribiyya” deg âss n 21 n sinyûr (février) xef Ufqir d tirmit (tentative) n umlilley (renversement, coup d'Etat) i tugha ira ad yegg. Qao tessawal u ttari xef Ufqir zund (am) ca n wamzviw (monstre) isverrdeven iwdan (hommes) ddaren. S uyad ilul (naître) d umyey n Ufqir u yemmarni (être ajouté) ghar umyeyen zi yeccur umezruy n Lmughrib. Yetteffu d Ufqir d netta d amasay (responsable) n qao min yeddar Lmughrib d adenkey (despotisme) d timekridvin (violations) n uzref n ufgan. May (min) d anegh isnezvmayen (préoccuper, intéresser, importer) deg wmyey ad n Ufqir maci d asezrey (justification) n udenkey d timekridvin n uzref n ufgan. Min itxataren deg umyey n Ufqir d tmazight i yesduggam (symboliser) Ufqir aked (au sens de 'selon') umyeye nnes. Winna (inni = ceux) d issufghen umyey n Ufqir ran ad sseknen is tmazight ur ttejji ghas inekbadven (criminels) am (zund) Ufqir. Aswi (but) nsen ur yelli d Ufqir wapa. Yella ula d tmazight. Maca pa qqa didewin (vérités) i tefsar 'Lahdat Lmughribiyya' ticcent ijj n wewlaf (image) yadvnin xef Ufqir. Tesfugg (faire couvrir de honte) iserkas (menteurs) u tessizzef (littéralement c'est déshabiller = mettre à nu le mythe) umyey n Ufqir u tsekan d is Ufqir tugha ghar s afrak (conscience) s isurdan (accusations) i d as yesseslegh (coller) anpil (régime) n Lhasan wis sin huma ad issuref (innocenter) ixf nnes s timekridvin (violations) n uzref n ufgan i yessen Lmughrib di tit (époque) nni. Maca aoeqqa azewwar (le noyau dur) n umyey n Ufqir d tamenghiwt n Ben Barka i d as sseslghan. Tamenghiwt ad tegga zay s ijj n unekbadv (criminel) d axatar itekkedv (hait) it' ughref amughrabi. S uyad yeghtes (décider) Ufqir ad ili taghawsa tamaezwarut i ghad yegg awarni umlulley d asuref (le fait d'innocenter) n ixf nnes s turda n tmenghiwt n Ben Barka. I weswi yad i tugha yesmun qao anzaten ingawiyen (Preuves matériel) i yessenzan (prouver) asaruf nnes u yessumanen (montrer) inekbadven yenghin s tidett Ben Barka. Max sseghden (se taire) ighmisen (journaux) n ikabaren xef tidedwin ad tixatarin i d tessufegh 'Lahdat Lmaghribiyya'? Max ula d ijjen ur izemmer ad isbugget (démentir) tidedwin ad? Acku qao tugha ssnen is akw ayad d tidett. Unna gh ad yesbuggten tidett ad yili zund wen ittinin is tammurt ur ttennedv xef tfuct. Attvas i ssiwlen xef ibekkadven i ur yeggi Ufqir ursar (oemmars). Maca oemmars ur ssiwlen xef ubekkadv n tidett i yetwabekdven (être commis) di tarwa nnes umi ten ixemmer ugellid s adu tmurt ujar ssimraw (vingt) n iseggusa. Umyey n Ufqir, am umyeyen n umzruy nnegh, yessentam (affirmer) is amezruy nnegh, zeg wkcam n lislam ar dghi, netta qao d umyeyen nlemmed iten deg wgherbaz. Awarni yal n umyey zeg umyeyen ad, yella uselluxcu (déformation) d usentil n tidett iteqqen ghar tmazight d imazighen. Aselluxcu yad d usentil yad n tidett tarran imazighen ightas (victimes) d imelkadven (wenni yeccaten s ulekkudv = bourreau), u imelkadven nsen imdaden (véritables) d tightas. Maca ttinin is asghun n userkes (mensonge) d aqudvadv. 2002 Tawiza. .
