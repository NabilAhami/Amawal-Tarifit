 48, (Avril 2001) Tamazight Tanfust: ANDER N YIMAN Tura t: Mayssa Racida Tandetc teghza, fardven tet, ssuydan tet, ssarsen as x ughezdis ssedjah d waman d oûrric n ucar. Tuss d nettat tufa tandetc nnes, tbedd as x ubadu txezzvar deg ixf nnes di tisi n tendetc temhedv, tuca ak imettvan unndven as d manis mma, x yeffus, x zvermedv, awarn as, zzat s, tuca ak tmûrt tedvsva (rire) s yict n tummart tudjegh. Tghaca t tayri yennufesren deg wemsagar. Tilullucin arxunt i tmajjut (senteur) axmi d ussan n tefsut. Ijdvadv tghennajen reghnuj nsen yecnan. Marra min d as d yundven d raggûj d radas itecna, iyarredv azvri. Tuca aked tazvyudvi n tudart, aked wehsin nnes yeddarrao as aked ighadjen nnes arezvmen swizvdven d ghar s, tuca aked tiri n tirawt (réalité) d useynu n tiddet d tayyut n tayri ssfayen startaren day s temfeyyadven xaf s. Tessulef (dessiner) ict n tadvsva x wudem nnes, udem nni i xaf seyhen wazghat d farwazghat, udem nni umi sserden ifassen n zman a iharcawen. Tessemghar tadvsva nnes, tarni tedvsva tewseq d day s ca n tkemmicin n isseggussa d ibarcanen tarra deg ûraq nsen yennedvni d icemraren, tessufegh d day s ioûrrac n iseghwan mxumbren mxarwadven iceddan day sen, tezregh d akid s tizemmujin n wussan tca tettu ten tawessant war d asen teggi tittvant (non pris en compte) weddvaren as jar ubarru nni yweddvaren da d dipa, tadvsva nni yuc it id ij n wur irettex s tummart ineddep tudart itgewwad it s ifiran n waregh yehnundven di tmexsa deg wbrid isseghden iwatva. Teswizzedv ghar ssedjah tessars idt x tendetc teddar it tnettiyi t amen txezzvar deg ixf nnes zzat s x ucar, tsettaf ssedjah nni teqqar: -" Mayemmi? mayemmi? " Tedvsva tarni d: -" Pa pa pa pa pa! i mara teddared anict i teddared yeodu d wanict gha d teddared oad tessufded tudart n buhber x henna m d tudart n buhber x wayyaw nnem, aseqsi yetghima d aseqsi, yetmuttuy d aseqsi, ytembeddar d aseqsi! zman nnem yemmurdvesv jar ifassen n zman a, taycit nnem tareqqaght iherwi t yidva yaharcaw, abrid n isvuraf nnem yujur iten tmarri d wapnaci, terja ghar m x uraji n tmeddit yuss d uzir iharwec zzay sent azal (valeur), ixarrisven i cem izzugharen ghar zzat ssedren war d fruryen, ggin ixamriwen tarsa xaf sen toejjajt deg wxxam nni i cem izedghen." Tsettaf ssedjah ijjen x aghezdis n yijj, mermi mma tessars icten tesrira t s rxezvrat tessawar akid s tenna s: -"Tudart a d ict n tebrat zeg wfus ghar ufus tmuttuy, ijj n ufus yegdvar, ijje n yezded, ijje n ghar s smus (5) n idvewdvan, ijjen war ghar s ca, tabrat a d adrar x weorûr mkur ijjen yecsi t war t yesrusi oemmars, wenni yzemmaren i webrid itawedv ghar uneggar, wenni war izemmaren yetghima d aneodvub, i mara yabrid a d tsawent x tennedvni, unudv x wennedvni, azray x wennedvni, mammec ggin isvuraf itxessvan abrid a yazirar mammec ggint tzemmar gha d inedpen abrid a yessahharen, abrid a d ict n tmeddit n tadjest tiriw tezzuggart timmar n tfawt tmûrudent day s mermi mma iwdvent arennint ghar zzat tettent akudv teffzvent, twara teghzvazven t, twara ssergadven t, mani mma swizzvidvent deg ûddar nni abarcan tweddvarent, yudrus mani tafent ixfawen nsent. Maca iri weddvarent. Tara nni n tmexsa deg ur nnem i d itfeyyadven s taycit n ulellec d tayri itriqriqen, swin zzay s ireghman, swin zzay s iwdan, swin zzay s wussan, jjin day s razagh d fad I yesghemyen Buydunan itejju iteqqes iccat s usennan, jjin day s tayarzawt (regret) d wayhat (tristesse) tmughzvuren deg wndrar n weshissef teggen twara x usarwet day s". Tessars ssedjah aneggaru, tedra tandetc nni qao s ssedjah tbeqdvi t s tcûrt, tarezzu x tizza day s thennac itent teddeqqas amen teqqar: "Mkur amsagar yedvfar idt wemsebdvi amen mkur asenti yedvfar idt uneggaru, mkur temzvi tmeqqar i tewsar, ruxa necc d cem ntemsafadv mahend necc ad dvfaregh aneggar ad as mgharegh i tewsar, cem darreo i wcar ad cem issaghri uweddvar.......
