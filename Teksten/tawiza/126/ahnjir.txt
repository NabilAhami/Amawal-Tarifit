 uïïun 126, Mrayur 2957 (Octobre 2007) Tamazight Iseqsan n tasekla n uhenjar amazigh di arrif Yura t: Said Belgharbi Tira di tsekla n imezvyanen d ijj n ughendef yemghar s wattvas, minzi ixessva ad yedwer umarir ura d netta d amezvyan, ad yendvar tbanta n temghar nnes ad yared aoban n temzvi nnes, ad t id isduqqezv d iskilen x tefray huma ad i zemmar ad yessiwedv min yexs ad yini n tebratin i yehenjaren imezvyanen. Di arrif oad iyyar n tsekla d tira i whenjar amezvyan d ayujir s wattvas, ra deg isefra nigh di tinfas nigh deg umezgun nigh di tusna n riraret... oad ntaf araq nnes yehsvar s wattvas. D manaya i d anegh itejjan ad negg ijj n uraghi i marra imariren d yenni i yisin imunas n umezvyan nigh yenni yarrin taynit i tira s tmazight ghar uxarresv deg ahenjar amazigh, d uswengem di min d ghar s d yeqnen n tghawsiwin neghnit i twattun zzat i wsugi d i tegg utilifizyun ghar ixf n yexxamen nnegh d min d ikessi n wefradv zi tzubayin n tusnin n midden. Zzat iwseymi n imawlan, s ughir xaf negh ixesv anegh ad nar taynit mahend zic, huma war ntghimi niggwej xaf s netta yeggwej xaf negh. Zeg imxumbar i di ntneqradv di tudart nnegh ntaf udrusent ghar negh timesmunin yarrin taynit i tesghart n uhanjar amazigh, tasghart nni i yeqnen ghar ufgan. Mara llan din imxumbar am yina ad yiri uhenjar amezvyan di tamezgha iweddvar s wattvas x tmmurt nnes waxxa netta d mmi s nigh d yedji s n tamezgha s tmaggit. Tanya ixesv ad nessen belli mkur ijjen zzay negh di radjagh nnes izdegh day s uhenjar nni amezvyan i tugha yedja, u mkur ijjen i teddar akid s uhenjar nni di taddart nigh di dcar. Imsermad n umadval yarezzun di tusna d umeddur d tagumi n imezyanen iwdven ghar ijj n wufugh yeqqar belli imar n waydud war itseggim ixf nnes huma iseggem zi temzvi n imezyanen nnes. Ahenjar amezyan d wenni d tfawet n tiwecca n tmazirt nnegh, agherbaz ura d netta iddakwar ghar s ijj n dqer yemghar di tasyumi n umezvyan, maca di arrif nigh deg imucan mani djan imazighen ntaf agherbaz yirar s wattvas di taghrusvi d ujeyyef n tusna n tmazighet yeqnen jar umezyan d tudaret i teddar. Zeg wami yeffegh uhewwas zi tmurt n lmaghrib agherbaz nnegh oad war (ibeddel) tasertit nnes aked uhenjar amezvyan, ntaf min tugha yedjan d amenzu oad d wenni iqqaren imehdvaren deg ugherbaz n wass a, agherbaz ameghrabi yeggwej x tamagit d umezruy d tutlayt d marra min yuyren ghar tudart n uhenjar amazigh di lmeghrib. Wenni ghar idvarnen tifray n idellisen ad yaf imeghrayen n idellisn a ssawaren x midden ujar zi min ssawaren x ixf nnegh neccin ayt bab n tamezgha, di lta di ntaf ayduden n umadval iwdven ghar ca n wufughen war ghar sen niwidv ca neccin di tmurt n lmaghrib. Ahenjar amezyan d wenni d reowetc n yimar nnegh, mara war das narri taynit ass a ad taf ad nardver tiwecca nnegh s ufuss nnegh, rexdenni ad nawedv ghar ijj n yimar ad anegh day s tagh tyarzawt.
