 56, (Décembre 2001) Tamazight Tamessizwarut: Ddvapir n tghennawt n ugherbaz ageldan i yidles amazigh (Sur le Dahir créant l’institut royal de la culture amazighe) Yura t: Muhemmed Budpan Taghennawt (création) n ugherbaz (au sens de l’“institut”) ageldan (royal) i tmazight itgenses (représenter), deg wmezruy amughrabi atrar (moderne), ict n tagrawla d takubirnikit (révolution copernicienne) s tidett (véritable). Taghennawt n ugherbaz a d ijj n wubbuy (rupture) aked ict n rxezvrat (vision) tugel (erronée) ghar tmazight d tnettit (identité) n Lmughri; tessenta (fonder) oawed ijj n weswingem d amaynu u d amdaddet (correct, juste) di tmazight u di Lmughrib. 1 - Amedwan (conception) yuglen (erroné = “izeglen” aussi) amezwaru tugha ixezzvar ghar tuggut (au sens de “pluralisme”) tadelsant u tirsyant (linguistique) am ijj n waggaz (menace = “anewwu” aussi) i tayunt tanamurt (unité nationale). Idd (quant à..) Ddvapir i yeghnan (créer) Agherbaz n tmazight, yessemlilley (inverser) amedwan a, am di “tagrawla takubirnikit”, mani yenna is asikez (reconnaissance) zi tuggut tadelsant d tirsyant yessedwas (renforcer) tayunt tanamurt. 2 - Tugha tideghrin (élites) timurabiyin d ikabaren isertiyen teddaren ijj n wemyaba (contradiction = “tanemgalt” aussi) yuman (manifeste): tettaren tadimuqratit, maca war texsen (tran) ad tiri s twagit (refus) nsen n tuggut tadelsant d tirsyant yellan d tumast (essence) n tadimuqratit tamdaddet. Ddvapir n tghennawt n ugherbaz n tmazight iteqqen tnukra s tmazight ghar tuska (construction) n tadimuqratit. Atpellu di tmazight d tafada (condition) n usilew (réalisation) n tadimuqratit. 3 - Tugha atpellu di taorabt d usetmi (marginalisation) n tmazight yessuncuk (constituer) ijj n waddad (attitude) day s attvas n tanmeglit (inégalité) arendad (vis-à-vis) i tmazight d ighermanen (citoyens) imazighen. Ddvapir n ugherbaz n tmazight yettini is atpellu d tugguzvt (soin, sollicitude) di tmazight itadef deg wsuger (garantie) n tugdut n isulufen (égalité des chances) jar imazighen d aoraben. 4 - Awarni ujar n 30 n iseggusa n weghnas (lutte, militantisme), war tuci tnebbadvt i tmazight ghas “linoac” () d “listinas” (). Ddvapir n ugherbaz n tmazight yezrey (dépasser) “linoac” d “listinas” u yegga s tmazight ict n tagahwsa d taghelnawt (nationale = “tanamurt” aussi) ad ghar s yiri ansa nnes deg wsermed, deg wmsawadv (communication), di talghut (information), deg wsneflulu anazvur (artistique). Zeg wghezdis nnidven, Ddvapir n ugherbaz amazigh, yesrusa attvas n iseqsiten: 1 - Agherbaz ageldan n tmazight yeghna dt u ytedds (organiser) idt Ddvapr, u maci asadvuf (loi) nigh tanadvt (décret). Ma tedwer tmazigh zi tghawsiwin n tanaya (souveraineté) i war iteqqnen qao ghar tnebadvt? Manaya ma d aglam (privilège) i tmazight nigh d tasiweght (entrave) i ticri nnes ghar zzat? 2 - Ma tetwazri (être dépassé) tnegmit (revendication) n usmendvu (constitutionnalisation) n tmazight awarni tghennawt n ugherbaz amazigh? Nigh asmendvu n tmazight yedwer d ict n tidett war ttaser (avoir besoin) amsegdal (discussion)? 3 - Meqqar Ddvapir war yidar tawalt n “lapajat” (tantala), tugha yuccel (foir) ad yessiwer di talgamt (liste) n ismigilen (missions) n ugherbaz xef wesmuni (unification) n tmazight s yijj n wudem yefra (de façon claire). 4 - Yesfada (requérir comme condition) Ddvapir deg inni gh ad yezzirwen (travailler) deg wgherbaz n tmazight ad ili ghar sen taghara (compétence) tadelsant. Maca war yesfad ca ad ssnen tmazight. Ma nzemmar ad nessedwen (imaginer) ijj n ugherbaz n taorabt ad irin imarezzuten (chercheurs) nnes war ssinen taorabt!? 5 - Ddvapir war yeghtis (tnancher) di tmesract n usekkil zi gh ad tetwari tmazight. Yejja manya i wgherbaz. Tamesract n usekkil d taxatart cigan u adghar (choix) n usekkil netta gh ad ygucclen (déterminer) imal (avenir) n tmazight u ad yesfukker (dévoiler) tanemlalt (les bonnes intentions) nigh taghdusit (les mauvaises intentions) n ayt bab n tghuni d warzvam (ceux qui nouent et dénouent = décideurs) arendad i tmazight. Imara yetwadgher usekkil alatini ad yili imal ghar tmazight. Uma adghar n usekkil aorab aqa ad yili d asenti n tmenghiwt n tmazight di Lmughrib. 2002 Tawiza. .
