 Uttvun 81, Yenyûr 2004 (Janvier 2004) Tamazight Tamessizwarut: Melmi ghad twasizvemzven ineghlafen nnegh imezwura n ulmud? (Quand jugera-t-on nos ex-ministres de l'enseignement?) Yura t: Muhemmed Budpan Dinni, deg ûssan a, ij (yan) n uzvmazv (Procès, 'Asvaradv" aussi) dat tasenbadvt (cour, tribunal) n teghdemt n tagduda (Cour de Justice de la République). D ta d tasenbadvt tamazzagt (être spécialisé) deg wsizvemzv (asvaradv = jugement) n ineghlafen d imasayen ixataren n uwanek (Etat). Asentel (objet) n uzvmazv ad d tamasit (responsabilité) n ineghlafen n tayerza (au sens de 'agriculture') di tamettant (mort) n ca n yinni (winna) yengha weysum n "ifunasen ibupaliyen" (vaches folles). Acku tasenbadvt tetwamdel (être convaincu, "tenzva" aussi) is amentil (cause) n tmettant n ightas (victimes) n "tubbupelya" (folie) n ifunasen d tarca (aliment du bétail) i ccin ifunasen nni mani (adday = quand) tughayen d iyenduzen d ibezzeotuten (imezzvyanen). Dix ineghlafen n tayarza d nitnin d imasayen xef tmettant ad acku ur hdvin, ur dvfan (contrôler) tarca i tugha yetwafkan (yetwucan) i y funasen. Ghar negh di Lmughrib, tezrey (teodu) ict n temsirt am (zund) tenni n "ifunasen ibbupaliyen" di Fransa. Tamsirt ad d iduqqzven imessawafen (terroristes) n Kazablanka di 16 semyur 2003 yeodan (izeryen). Imessawafen nni isduqzev ixf nsen huma (afad) ad nghen attvas n iwdan, ur izemmar ad ilin ghas (mghar) d imadvan (malades) s yat n tubbupelya tujar tenni n ifunasen n Fransa. Wi yellan, dix, d amasay xef tmettant yesmentel (causer, provoquer) uduqqezv n imessawafen ad? Tarezzut (asiggel) xef westan (réponse) i wseqsi ya itawi ghar yan n useqsi nnidven: wi yellan d amasay xef "tubbupelya" n ihudriyen (jeunes) ad, "tibbupelya" yad i tugha d amentil n tassart (fait d'avoir lieu) n tmettant d tightas? Am ntaf amentil n "tebbupelya" n ifunasen di tghawsa itsezvmizv tsenbadvt di Fransa d tarca tameggufsut (Mauvais) i ccin ifunasen nni adday tughayen d imezvynen, i ntaf amentil n "tubbupelya" n imessawafen n 16 semyur d "tarca" i ccin mani tughayen d imezvyanen. Imala (si) ineghlafen n tayarza di Fransa d nitni d imasayen xef tawsit (type, marque, 'anaw' aussi) tamezwarut n tarca i yucin tibbupelya n ifunasen, ineghlafen n ulmud (enseignement) di Lmughrib d nitni d imasayern xef tawsit wis sin n tarca i yucin tibbupelya n utarras (afgan) di Lmughrib, tibbupelya ya i nezvra (nennay) tilemtin (échantillons) zzay s ass n 16 semyur. Tumayt (contenu) n tarca iccin imessawfen n 16 semyur d tussna yeggufsen, d tiktyin (idées) yexxun (être mauvis) d ipilen (programmes) ighurbizen (scolaires) i tezzvun taweppabit (wahhâbisme) deg imehdvaren… Awarni 16 semyur qao idvewdvan scanen ghar uneghlaf n tghawsiwin tijjdant (affaires religieuses) is d netta d amasay xef min imsaren ass 16 semyur. Maca inni issurdan (accuser) aneghlaf n tghawsiwin tijjdant ttun amasay amdaddet (véritable) xef uneflulu (apparition, naissance) n umussu (mouvement) islamawi manis d iffegh usiwef (terrorisme) i yewtin ass n 16 semyur. Amasay a d aneghlaf n uselmed i tugha "yesseccan" (faire manger) i ynelmaden (imehdvaren) "tarca" d yusin zeg ugmudv (orient) aorab s qao min day s d aselmed n usiwef d uzuccen (incitation) xef wakkadv (haine) n yinni ur illin d oraben d tmenghiwt nsen. Tarselt (fondement, pilier) taxatart n uselmed a n usiwef d asoarreb aghrud (arabisation totale) n ulmud aked ineggura n iseggusa n 70. War (ur) d arrin imughrabiyen taynit i wmija (danger) n "tarca" i tetten inelmaden nnegh deg igherbazen ar (al) ass n 16 semyur. Man aneghlaf i d issidfen "tarca" tameggufsut yetwaskelwen (être empoisonné) ghar igherbazen mani t ccin/remden inelmaden nnegh? D Oezzddin Loiraqi ghar uneggar n 70 d umezwar n 80. S unelkem, d netta d amasay xef min yemsaren ass n 16 semyur. Dix, melmi ghad wetwasizvemzv Loiraqi am twasezvman ass ad ineghafen yejjin ifunasen di Fransa ad ccen tarca yeggufsen? 2002 Tawiza. .
