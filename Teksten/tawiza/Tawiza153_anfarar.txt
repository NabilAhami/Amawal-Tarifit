 uïïun 153, ynyur 2960 (Janvier 2010) Tamazight Tanfust: Anfarar Yura tt: Yusef Oazz Eddin*, Asughel: Said BELGHARBI Ghar usezghel n tsammart, nezzuyyet ad nkeyyef lkif zeg ict n tqedduht s ughamus yedla s arecwan, fessyen igarruten nnes s tufft n waman, mermi mma nekkes d zzay s yewt n ugarru ad itwarz jar ifassen nnegh am ighedj n ibawen!. Deghya neqqen xaf s, dja mermi mma nekkes zzays ca, staff it teccur i twarat nnidven, axmmi war zzay s yeffigh walu!!.. Nemxumbel di ticri nnegh, nweddar deg ict n tsedjya war ghar s amezwaru ura d ameggaru, smuqqlegh deg udem n umeddukel inu. Ssfeldegh deg uraq nni ghar ucemcem n snat n twessura, ssawalent s yiles n ayt bab n tmurt a, tutlayt nsent temsarwas deg iles nni i da jjigh wapli! Lehfent di reyzur d ibarcanen, beddent zzat i tewwart tameqrant n ucarcur. Nezli nec d umeddukel inu ghar zzat s kuz nigh s smuz n isvuraf xef uraq mani beddent snat nni n tmettvutin, idwel umeddukel inu mani tugha nella, dfaregh t nec, war nuci ar lami nuffa ixf nnegh deg ammas n ririwet n ict n teqseft d taqburt... nudeff xef texxamt mani yeqqim ijj n uzirar xef lkursi nnes, itissiqn s wedrim d toeqqayin n lluban..., itoini azirar nni d wenni d agellid n teqseft a, yeqqim isess, isekwwedv dey tmecdvahin d timezvyanin temraywant zzat nnes, teqdiddisent s toeddisin nsent, am trefsiwin ghar llgha d zrayar n wadjunen, iswizzedv umeddukel inu ghar ict n teghyact temghar, temmars deg ict n tzewdva n nnuqart, maca zzat ma ad tt id yeksi, temdareyyac teghyact nni jar idvewdvan nnes, tedwel am taneghda, izzuzzar usvemmidv deg wadvu. Narwer zi texxamt nni, nudef jar tixuxxatin n teqseft, u gha war yelli ibriden n wuffughen, nexs ad nacar tasenduqt nni n igarruten, maca imerdas deghya arxun i tghuyyitin d iraghan di marra timewwa n teqseft, deghya ntwacaref, netwattvef huma war ntrikkiwr ghar mani, war d anegh tessiwedv tazla nnegh ad nali xef usanen nni (escaliers) yuylen ghef igudar n teqseft!! Neqqim nsmuqqul deg imerdas yemsarwasen di tefraz n wayawya nsen. Nxezzar deg ijj n uwessar inya xef usanen nni s tazla am tizemmar n umezyan, umi yiwedv ghar ticcewt n ugdur, marren ca n isemghan iwyen d reqcuo n uwalef (usewwar). Ict n macina d ict n lkamira, deg umezwaru ksin anegh tiwalfin i marra idnegh, wi snat i nec d umeddukel inu, ghar umeggaru iswizzedv umwalef (lmusvewwir) ghar tasenduqt nni n igarruten issarsi t zzat nnes, uca yeksi yas s umarri ict n tesmaqqalt. Carfen anegh s isekraf. Gewden anegh ghar ict n tkurmut di teqseft, maca deghya nuffa ixf nnegh nteqqen ijjen ghar wennedvnit nec d umeddukel inu, nuffa ixf nnegh nemseyyah di tisedja, manis i d nudeff i twarat tamezwarut ghar ubaraz n teqseft. Deg ijj n wakud, ksin anegh ghar texxmat n uwessar nni, lami ghar s niwedv, deg awal nnes amezwaru isseqsa, inna negh: (Ma yemmi d tudffem ghar umezruy n teqseft inu, manis d tuymem taryazt zey ttusim ghar s, d mayen kenniw d icqan ghar s?...). Nessusem, war narri xef iseqsan nnes!... Netta yugzu i wesqar nnegh, yesvuref ghar tburjet n texxamt, isijj, itwara zzay s mghar tubbarcent tlehhef s test nnes i tburjet. Umi smuqlegh deg umeddukel inu uffigh ibeddel xaf i, yeqqen tifraz d timaynutin i wudem nnes! Arrigh taynit i wawal n uwessar nni, ssiryegh tittvawin inu, twarigh iswizzdv d ghar i s tqedduht nni yedlin s areccwan, yuffsen s igarruten yuffen, yeccur d awal nnes s tayri, uca yenna yi d: “Ssnegh ira tugha teggid ad tacared taqedduht a, mermi mma zzays teksed ca n ugarru itmarni d wennidvint d amaynu, s uyenni rexxu xsegh ad ac twardigh d tiwsi, ad ghar k teqqim i marra ussan”. War ucigh s minzi t snnemiregh di ltva nni, ura d min das nnigh d tamwarit, maca zeg akud nni war zrvigh ameddukel inu, nec uffigh ixf inu deg ict n texxamt tedla s test am tzeddight deg imedvlan, qqnen xaf i tiwwura i marra ussan, netta war ssinegh man tammurt it yeccin!. Asughel deg unebdu n 2959 * D amyaru d AKURDI. 2002 Tawiza. .
