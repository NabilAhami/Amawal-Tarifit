 uïïun 159, sayur 2960 (Juillet 2010) Tamazight Tazrawt (étude): Tasmaqalt n “tazwagt” deg izlan imazighen di Arif Yura t: Said BELGHARBI “Yardegh aqareftvan bu seboin n reqfur mami wami yuyar, yejja yi mantca deg wur.” Izri d arifi Amezruy n uneoreq deg umadval: Tazwagt (l’immigration) d amutti zeg udghar ghar unnidven, amezruy nnes issenta zi trarit n ufgan xef ukal a, mammec i nessen; war tedji ca d ict n tghawsa teqqen ghar bnadem wapa, ghar s tighuni ura ghar imuddar (animales), ura d nitni tmuttuyen zi tmurt ghar tennidven am ijdvidven deg ujenna am iserman deg ayl.. tneoraqen am ufgan zeg isekraf n tasmughi (l’esclavage) d tinokraf n tzawa d lazv d tarewra zi takritt (la violence) n imenghan n ufgan d tiwaghitin (disaster) n umnadv, mani tgajen arezzun xef ufra (paix), d uzeddugh n tcuni. Attvas n wayduden i yezrin tawada d ugaji zeg akal ghar tmurt nnidven, ghar negh deg umezruy n usghan (ddin; religion) attvas n tmacaputin, zzay sent tanfust n umazan (Prophète) “Musa” wami igajj yarwer zi taqsuhi d tighennanin n waydud Aferouni, d tmeddurt n umazan “Muhemmed” (tzit n arebbi xaf s) ura d netta igaj iked imselmen deg uzwag nnes axatar itwassnen deg umeddur n tmeslent; wami imutti zi temdint n Mekka ghar Lmadina, zi senni d usawen zi issentan imselmen tassakudt (datation) d uzemmem n asakud (date) nsen. - Amezruy n tazwagt di arif: Deg ammas n timidvi n simraw (Le XXe siècle) iyzeryen, awarni imenghi amedvlan amezwaru (1914-1918) d wis ssin (1939-1945), d isfuffyen iked unfarar nsen ict n tudart tudven s taqsuhi, tmukrist n umenghi temsar s wattvas di tmizar n Uliman d Fransa… mani uffin t ixf nsent ffghent zeg imxumbal n umcubbec d umenghi udfent ghar umukris n tagumi, mani tugha udrusen ifassen n tawuri, zi senni xarrsvent ad swizant s imeswuriyen (travailleurs) d ghar awyent zi tmura n tefriqt ufella d tefriqt tabarkant, zi tmizar n Lmerruk d Dzayer d Tunes d tmura n wadday n tnezruft am Ssinigal d Nijirya…, zi senni ssentan irifiyen tgajen d refrug ghar tmizar a n Urubba. S tmarrnit xef umussu n tazwagt ghar tmizar n Urubba, dja dinni aneoreq d ameqran n irifiyen i gajen ghar tmazirt n Dzayer, yenna “Ugustv Mulyiras” deg udlis nnes “Lmeghrib ig itwattun; taffi n arif”, deg ussegwass n 1895 ujar zi 20000 n irifiyen ssentan teddun mkur assegwass s rwacunat nsen ghar tamurt n Dzayar, arezzun xef tawuri di tibehhar d reorasiyat n Ifransisen i tugha izedghen tamurt a awarni uhewwes nsen. Azrawi amazigh war yettu ca ad yiri d ayt n uzemmem s tatlayt deg umezruy nnes s yezli amazigh i xemmren min izeryen zeg ussan nni, izlan di tezdegh taycint n umazigh d tmazight xef minzi tugha itaca n tiqqest idi tugha yezdegh umukris n uzwag deg ugensu d tmeddurt n tudart n irifiyen marra. Ijj n izli izemmem jar isekkilen nnes ijj n umezruy issawal xef tidet nni, wami tugha assif n Merwect d arefsiw; wenni it ghar izvwan ghar ujemmadvin axmi d irur d amaynu, nssefli d ghar tazrawit tarifit teqqar xef yiles nnes u nettat tessutar zi Merwect ad tcaref inagiten (Les inondations) nnes, huma war ttiwi llif nnes yeggin ad yezvwa ghar tmazirt n Dzayar: - A Merwect a Merwact; taqarqact n tezra Aqqa ad teghdared llif; axmi cem ghar yezvwa! Tazwagt a teslul d akid s attvas n inonkraf i yejjin imnneoraq uylen jar timessi n ugaji ghar timura n midden d tzawa d imxumbal n tmura nsen, taycint zi issemghar uzrawi d umedyaz amazigh s ridaret i yejja teddar deg izri arifi. Izli ya d taqessist di tiddi d taqudvadvt, maca imghar di remoani nnes, d takemmict n iwalen yufsen s timitar, izli i yeqqimen yeddar am imedyaten, itmuttuy jar tirewriwin n tniba deg ibarazen n tfugliwin d tmeghriwin, ixemmer jar tighit nnes tidet n ijj n wass s min tarbu xef uorur nnes n tizedt d tarzugi, izli d yusin d amazan (Prophète) am ijj n ureqqas n umezruy nni izeryen, issiwel ura xef tazwagt am ict n tmukrist iddar it mkur axxam d mkur dcar, teghza di tamektit n wur n mkur amazigh.. Issiwer urifi d trifit deg wattvas n twaratin xef remoani n yezri, ntesla ghar s yeqqar di sin n izlan a: - Ma tghired war fepmegh; i remoani n yizran Chal mma teghri d dinni wi cek yeodan - Izran n yidv a kurci s remoani Iteqes rhar; i wen war sen ifepmen ci. Adghar n tazwagt d tayri ghar ufgan arifi: Tayri d tazwagt truren d deg izri arifi am ikniwen, tazwagt d ict n umxumber imghar deg umeddur imazighen, attvas i tesyujel n wurawen i tugha i texsen ayawya nsen, attvas itessigwej n lwacunat xef idughar nsen, izli di ghar naff tazwagt ntaff dinni tayri tessdel, dinni amarret nni zi iytaca wargaz amazigh u netta i tegg idvaren nnes ad iniy deg ugharrabu, mani i tejja tiqqest deg wur n temghart nigh n tamiri nnes, tessaqriq ura d tinedwin n tittawin mani izdegh unessis n imettawen, s umsebdvi d lagwaj n ijj xef unnidven. Ntesla ghar urifi yeqqar deg izlan a, u netta i tettar zzay negh ad naca s taycitin nni yeggwin s tidet n tiqqest d uxeyyeq nni: - Umi truhed lxarij, rebbi war d ac isemmah tejjid rbar inu, deg ubrid iseyyeh. - Cek txemmimed i tnaya n macina nec txemmimegh wi ghar tejjed i fenna. - Tesghuy macina dexxan yegga qadus tenni yura Arebbi ima ad tnemha s ufuss. - Wi ghar i zi reodu zeg ugharrabu n Mritc yeksi yayi mami inu yejja yi d tayujitc. - War d as tsimihegh i tveyyara yedvwin Teksi ayi mami inu tiwi t ghar trumiyyin. - Tveyara tamcumt, tveyara a tamenoect tiwyed ayi mami inu tejjid ayi d tabuparect. - A Sitimi tuyar a Munqada raja ma war ghar m bu rexbar n llif ad ipajar. - Isghuy ugharrabu, yenna s zid ad tenyem mami leoziz inu gherben t imettvawen. - Teksi d llif inu ghar tmurt d tarumect aolik cem zrigh ad am qqnegh ticarrect! Zzat i uxeyyaq a di ntaff tayri tusligh (affixe) iked tazwagt, s uffsay negh i yezlan a, nfarren d zzay sen ijj n umawal n iwalen i yeqqnen ghar tghawsa n imesniyen (moyens de transport) am: Macina, Agharrabu, Taylalt (avion), Sitimi, Munqadva..), mammec i ntaff amedyaz amazigh issemras (usage) day sen ijj n umawal n uxeyyeq yuylen ghar: (ighuyyan, tuyyujelt (Orphelin), imettawen). Tamazirt n “Uliman” deg umussu n izlan n tazwagt: Tarbat tarifit tedwel tmmexsa marra min tt yejjin ad temsvebda xef llif nnes ad xaf s taggewj, tcarep ura d timizar nni mani yuyar llif nni, yeddu d tarewra zi tzawa ghar tawuri, huma ad ibeddel imal s ufuss nnes. Mara narzu deg izlan n tazwagt xef tmizar n midden, xef timura n Urubba mani igaj urifi, ntaff izli issawal attvas xef tmurt n Uliman, mani mmutyen abarru n irifiyen. Tamdyazt tarifit temsawal, temmnegh, temzawar u temghunnan iked tmurt n Uliman, i das yeksin Llif nnes, tucari t ghar uraq nnes, neqqar deg izlan a asutar nni tegga tmettut Tamazight i tmurt n Uliman mahend ad tewc tirelli i umennuoruq ad yedwer ghar tayri nnes; ghar ukal nnes: - Aya Alimaniya a ixf awen n uriri tugha yi d amesrem tarra yi d arumi. - Aya Alimaniya nec d roarr nnem, sarpas i llif inu; char inu oam nnem. - Aya Alimaniya ma tiggwej ma tudes, “temniyyam” n webrid d ugharrabu yeghdves. - Aya talimanct ma war d am yewci wur nnem jjit ad yarewep, char inu oam nnem. - Mara truhed Aliman wcayi tabrat inu nec war zemmaregh resnin a llif inu. - Aya musejjala am gharsvegh ifiran teksi d awar inu tiwyet ghar Uliman. Tazwagt tesrurd ijj n umarret di ulkisen (Psychologique) n urifi, mani tweddar tidet jar iseqsan, tedwer trifect tzwar amiri nnes t yejjin ad zzay s isxan, ad yetleo d zegwu ghar ujemmadvin, n tesra ghar tazrawit tarifit teqqar deg izlan di tessiwed akemmic n twalfin n tidet nni izedghen deg wul nnes: - Ma tezvwid lxarij wi ghar day i tejji d tejji d aduf inu deg uxxam itejmid? - Tenna yi mayemmi tetru d jjayi ad rugh nnit llif nnem yuyar wenni inu yedfari t. - Ad qessegh afuss inu a das arenyegh aqessi wami yezwa llif war das odiregh awessvi - llif inu yuyar, iruh ad isafar ggayi d di tebrat zeynayi d rxadvar. - Mara truhed lxarij ggayi di rbar nnec ggayi mani tegged mami ttesrih nnec. - Inni yi: ad am ggegh min war d am yeggi babam ad am ggegh ttesrih ad tezwidv Aliman. - Inni yi: “mani cem ghar ggegh, ugin lwalidin?” ggayi a mami inu d lisvanesv i tumubin Dinni u dinni uran n izran d tizlatin, attvas zzay sen cnun ten di tfuglatin n tmeghriwin, izlan a mani itemyidar waydud s ict n taycint tissiq d zzay sen am isefdvawen; tesrugha tifawin xef umeddur n waydud arifi; xef ulkisen nnes i di izemmar ad t issefsi d iwaren imquddan iked ayawya nnes, am n muquddan idurar n arif, ghar negh d itemsawalen xef umezruy nni n ijj n wass, amezruy issuren iked tazwagt d utlao, am ijj n umnus azwar nnes ighmi deg azghat nni, tidrin nnes oad ghemyent id d raxart i nedyen tifdar nnes deg ayl agracal (Méditerranéen), mani tmetta temzvi d ighedduten di tgharrabutin n unaruz (Espoir).
