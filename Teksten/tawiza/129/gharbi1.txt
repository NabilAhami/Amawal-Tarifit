 uïïun 129, ynyur 2958 (Janvier 2008) Tamazight Muhammed Xayr Eddin: Aseklan i temmexs raxart Yura t: Said BELGHARBI "Temzvi inu tezri jar idurar, d manawya u yurigh deg i dellisen inu, texsegh s wattvas tazeddight deg idurar nni, teqqen ayi tayri ghar tammurt a, texsegh ura d azvri n waman nni day s itazren, ijdvadv nni day s ittvawen, tisudvar d tsekwla nni day s ighemyen, … mani d ighemmi wargan, mani tedja ict n tudart war ghar s utcma s.." (M. Xayr Eddin) Muhammed Xayr Eddin irur assegwass n 1941 di dcar n (Azvru n Wadvu) deg ughezdvis n temdint n Tafrawt d yusin deg wadday n tmurt n Lmerruk. Deg wehsin n tamazirt a ntaf yuyer ghar s ijj n umezruy issawar s taycitin, deg uraq a mani d trur tasekla aked tamment id itazren zeg irsawen n imazighen. Tasekla ya igha war djin idellisen di ghar twaffar am tsekla n midden, maca oad teqqim teddar waxxa amenni, teddar deg izerfan n imezdagh di tussna n warudv nsen, di rfadj deg ighennijen n ahwac deg usefru yesdaren x imiten n tiniba, deg iraghan n tamawayt i zi ssawaren idurar… zi marra tarawin a i zi yeswa umarir d uzrawi amazigh Muhammed Xayr Eddin tasekla nnes u oad war yenghi fad nnes… Umi yiwedv di buhber nnes 23 n issegwussa yefsar asefru nnes amezwru sadu uzwel (Aprudji d abarcan), zi senni deg ijj n wass zeg ussan nnes imarzuga yegga tizemmar zi ghar inegh ixef nnes, maca raxart temmexsit. Deg useggwass n 1961 arami d 1963 yudef amadval n tawuri, isenneta ixeddem deg ijj n rfusina di temdint n Ugadir. Di ltva ya tamdint n Ugadir tughat oad ttaca iked tiqqest d isendifen i day s tejja tarjijat nni tameqrant das imsaren, di ltva ya tugha ura d ijj n umeghrabi war izemmar ad din izdegh s tiggwdi din tejja tarjijat nni, maca Xayr Eddin yixdar itt d tamdint nnes waxxa tugha tezdegh day s raxart. Deg useggwass n 1965 inneoareq ghar tmurt n Fransa, mani iyruh ad yarzu x ixf nnes jar ijj n umadval itett ayt bab nnes, di tmurt n uzewegg immutti Xayr Eddin ghar iyyar n tira, itari ixf nnes s tutlayt n tefransist, yura amezruy n tudart n imazighen d mammec teddaren di radjagh n idurar n Watvlas. Tira nnes msebdvant x wattvas, tira urment s taqsuhi n usefsi nsent, yurem itent s imunas d ifiran n tayyut, yarra day sent taynit i marra min itwattun di tudart n imazighen. Zeg idellisen nnes i yura ntaf: (Agadir), (Tudart, turja, aydud deg weddar nnes), (Tarzvugi inu), (Amuyi d Tudart n Agencic), (Aghemmuy n tlullucin yemuzzaren), (Tanfust n ijj n akuc)… idellisen a smunen jar asefru d ungal... d wattvas n tira nneghnit itwafsaren di tesghunin d tineghmisin di tmurt n Lmerruk d tmurt n Fransa. Yenni igg isnnen M. Xayr Eddin zi radas, qqaren tugha iteddar ijj n umadval nnes, axminni t itadef yarecm it s ixf nnes iteddar day s mammec yexs netta, s uyenni id tasentt tira nnes ufsent s tamuyin (Légendes) d tifras n imazighen. Tameddurt n Muhammed Xayr Eddin teccur s ughendef d tarezzut n utarras deg ixef nnes iweddaren jar ixfawen nneghnit, maca Xayr Eddin itghima d ijj n wedrar zeg idurar n tasekla n Tamezgha zzat i Muhammed Cukri, d Katib Yasin d Mulud Moemmri… d wattvas n iseklawiyen imazighen yixdvaren ad arin ixef nsen d waydud nsen s tutellayt d yires n teoraft nigh n tefransist. Deg useggwass n 1995 tucar it raxart ghar uraq nnes, yemmut u netta d amepruc, maca raxart nni war tzemmar ad tacar tira d tsekla nnes zi twangitin n imeghray nnes.
