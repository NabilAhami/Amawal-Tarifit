 uïïun 129, ynyur 2958 (Janvier 2008) Tamazight Tanfust: Axxam n tirra Yura t: Mimun Amsbrid "FIAT", "SEAT", "PEUGEOT", "OPEL", "GOLF", "CITROEN", "AUDI", "FORD", "MERCEDES"… Tibhrurin n ikamyuten d titumubilin: kuy tamitart s tbhrurt nnes. Di fabrika n Baku, di tawrirt n Balaw, taghnast n Barsiluna, Badi u Tcuket, Manwilu zi Andalucia, Roberto zi Galisia, Hemmu zeg Iyyar n Tzaght d innidven haryen ghuma itwandaren hama ad zzays ggen tibhrurin n ikamyuten d titumubilin. Ttawedv d d isumadv n icudvadv n ghuma qqarsen, issekka ten Badi di tsirt twara awarni tennidven ar dewren d taneghda d tabarcant. Itas d Manwilu itccar zzays tizyawin, ygger itent di tsirt n yiggwa. Manwilu ghars tizmmar n uynduz n Kurrida: ur itiher, ur itxyyiq, ur issiwir, ur idvessa. Tittvawin nnes ur tksent zeg imi n tkuccet: ihdva arecti n ghuma i d itffghen d ijartar d ibarcanen nndven, iteffegh zzaysen ireggi, ssecmadven. Yssizzara sen Manwilu ighadjen; twettvan d jar ighadjen d idmaren nnes. Ifessar iten x ict n tdabut n wuzzar d tazeyrart ca 10 n isvuraf. Badi u Tcuket, s yij n uyarcuh anict n ughir, issuyedda icnifen ad n ghuma, ibttva ten d tisqqar, issigha ten i Rubirtu i ten gha yarren d tibhrar uydant. Ijartar n ghuma d imuzzuren, ur ittini Badi ar daysen issek ayarcuh nnes. Ussan imzwura yuca aked tighardvin nnes wdvant id; ighadjen nnes uffen, twzwizen… Di fabrika n Baku tirghi ttawedv 50° d tsawent; tidi tnessas akidsen; aman ur tarran afad. Ass amezwaru, Badi itwarna; usin as d imettvawen: iffer udem nnes, izvemm d tittvawin, imettvawen ssvuren akid tidi. Manwilu iskkwedv ghar Badi s ughzdvis: ur as iccûr tittvawin maca ûrdji s wawaren i das t' itini. Oemmars Badi ur izvri tamara amacnaw ten i yufa di fabrika n Baku: ikrez, imjer, yeksi aqbub, igga taffiwin, isbedd ituma, irzva imura oemmars ur ikrizen, yuyem, yisi acar i tzghwin n taddart nsen… D tidet, dja itaher, maca isnuyfa; awarni umcri itarra azedjif nnes; ur itegg ghas min izemmer ad yegg; min ur yiwidv ass ad itjja t' i tiwecca. Tamara nni dja itaca akids d tmizidvt. Ghar ughlluy n tfuct, xmi ghad yndar tittvawin xf iyyar i mjren netta d rwacun nnes, ad itwara icuwwafen n imendi bedden, itmeysi wfud nnes, awhar ittvaw; iddakkûar amxmi ommars ur imjir. Amenni xmi isnufsur tyuya awarni ij n wass d azeyrar n tyarza, ad iswedv awarnas, ad itwara tamurt icarzen d tazggûght nigh d tabarcant s timegga n ufus nnes, itaca akid tacbabt nni n tamara ysmridven i arrimet tfessi cway-cway ar tdwer d abirrep, d tumart… Da, di fabrika n Baku, tamara ghars tinbi n yighghed. Jar ighudar imarghighden d macinat timarghighdin d taneghda tabarcant itccarren imzzughen d tittvawin d tarrut, tuzjuwt n ghuma icemdven – itghridjdjit wur i Badi; tasent as d tmrudjdjay. Itexs ad yawedv tamurt, ad isnuyfa maca war izemmer. Iteggêd ad t' izvar "uxifi" (= anbdad) ad t' yessufegh. Ghar tmeddit, netta ad yeqqen tittvawin d macinat n Baku ad ghars beddent d isidven, d imzviwen, d ifisen arzzmen ghars aqemmum, swazzadven ghars s ighadjen d idvaren d iyenfifen; tazzren awarnas ad t' ccen. Ittrwer, itazzer jar ighudar, ineggef, tidi ttpnunnuy akids… Netta iggûr ad itwattvef, netta ad ikker zeg idves. Itaf ixf nnes iswa s tidi, yuher, idmaren nnes tsudven. Itkker, iteg cway n wghrum d ict n tnakat n watun deg wqrab uca iggûr ghar fabrika n Baku. Ict n twara ismuttvy it' Baku ghar ict n twuri nnidven; ict n twuri tfsus, ur days bu tamara attvas: ireccem ismawen n tmitar n ikamyuten d titumubilin x tbhrurin: s umellal idveffer ihnnicen itwawten x tbhrurin di macina… Umi yufa ixf nnes deg wxxam ad n tirra yuca akid ca itekker d days: iskkilen imellalen n "SIMCA" d "FIAT" d "FORD" d "PEUGEOT" d "AUDI" d "TOYOTA" d "MITSUBISHI"… tnufsuren zi tbhrurin n ghuma, teffghen zeg ismawen n ikamyuten d titumubilin hama ad dewren d tirra – tirra n tidet deg wzdjif n Badi! Tirra nni dja di yzdegh u zdghent days zzat ad yas ghar Katalunya; zzat ad yssen fabrika n Baku… Tirra di Fabrika n Baku!! Zi mermi ur izvri tirra? Tirra i dja iccûren tudart nnes di Tmurt urant t' zegwami izvwa ghar Sbbanya. Dewrent deg iman nnes amecnaw tiggaz nni yccmechen akid isggûsa. Umi, deg "wxxam n tirra" n fabrika n Baku, issenta ireccem ihnnicen s umllal hama ad tafed tmwaran, ssntant tirra nni dja ittvsen deg wadjagh n Badi tnuhmuzent, zruzant xafsnt takka: awarni iskkilen yudrusen n ismawen n ikamyuten d titumubilin, Badi iqqer ca n tirra nnidven; tirra nni xf dja itrujdud, dja t' itejjan itazzer x yidvs hama ad zzaysnt yarnu… Ass di yssen is asghun nni dja t' yqqnen ghar tirra iqdvu, ass di yssen is tarezzut xf wghrum ad tssiggêj x tirra, yuca akid ixf nnes d ayujir. Ajummi n tirra ydder it' Badi d aseghres, d abttvu n yxf x ixf nnes. Badi ommars ur ynki tirra, ur tent ittu, ur tent ynghi deg ixf nnes. D tidet ydr itent, yffer itent deg wadjagh n iman nnes, maca ommars ur days mmutent. Mermi ma izvra ca n uftwiz n tirra itrezza xafs ujummi, itkker d xafs uyenni… Amya, umi yssenta yareccem ismawen n ikamyuten d titumubilin deg wxxam n tirra di fabrika n Baku, irin iskkilen zi twaggen ismawen nni tnufsuren ayad zeg wyad, tmunen ayad akid uyad di ca n tirra nnidven: tirra nni ittvsent deg wadjagh n Badi. Ihennicen nni idvffer urdjin d inni itwarrin di tbhrurin n ghuma: Badi idveffer ihnnicen n ca tirra nnidven… Tirra nni umi yewca azyen n tudart nnes. Ihnnicen n tbhrurin n ikamyuten n fabrika n Baku dewren adu tittvawin nnes d tirra nni dja xafs iccûren tudart s wazvri, s usqsi, s twajit, s uwddar d twafit…: tirra n Burgis* d Blancu* yggin am ticri di tayyut; tirra n Yursinar* yggin am tacukulat deg wqmmum; tirra di munent tizidv d tighit n Bart*; tirra n Dirrida* yggin am ticri di tsawent; tirra n Ibn Oarabi* d itawyen annay; tirra n Nitc* ikessin afud, igharsen ajenna… Tibhrurin n Baku dewrent d arraten, d idellisen nndven i Badi zi marra ighzdisen. Amzri n tirra issettu yas mani idja. Ittu Baku, ittu Manwilu, ittu Hemmu. Ittu Tuyuta, Mitsubishi, Furd, Sitrwin… Ittu ixf nnes. Yzzvu azedjif di tirra nni ur itmwirin: Tirra nnes. Ur d yarri taynit arami yuca akid Baku itacem days: "Uyur ad ac wcegh min t' ttarsed; tiwecca ur d ddakkûr". (Mimun Amsbrid) 2002 Tawiza. .
