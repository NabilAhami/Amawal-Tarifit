 uïïun 177, ynyur 2962 (Janvier 2012) Tamazight Arrud n wmrabedv Yura t: Mimun Amsbrid umi tarzm tawwart n wmrabedv hama ad tzzerz taneghda n ucar x wndrv nns, ad tssk tasttva amen ttgg zi mermi, Mamma n Mmuh u Heddu tennedv zzays tmurt, ikka xafs uynni, tewdva x wndrv, tkka ssin d abrid. D Hmmut n Mezzvyan, d yusin ad tawi ccmeo i wmrabedv, i t din yufin tqbar x wndrv. X umzwaru ttghir tmmut. Umi t id ghars tksi tufa oad days tudart. Tuzzr ghar tmszizdct, tuym d ict n tjra n waman d ismmadvn, turra s tn x uzllif. Tzzerz Mamma n Mmuh azllif nns, tarzm tittawin – maca ur tssin mani tella. Tamghart a i ttugha zi rbda d tawssart, ur itwassen mayn ghars n isggwusa, tugha tudart nns ttennedv x wmrabedv: tzruza xafs taneghda, tssekka tasettva, tessadaf timzyar... ur ghars bu tudart barra n wmrabedv: ur ghars axxam, ur ghars rwacun – axxam nns, arwacun nns, tudart nns tsmun -i-tn gw mrabedv. Wi yksn arrud x wmrabedv? I wmrabedv mayemmi ur as ynghi ifassen i wnni xafs iksn arrud nnes?! Mayemmi ur t’ iqdviw zegw aorur?! Iwdan tnduddu zzaysn tmurt: mamec yegga wnni yjjit’ wur nns ad yegg mayn yegga i wmrabedv: amrabedv umi tnndven hama ad ttaren anzvar xmi tewta tzawa; netta i ghar ttrunt tmgharin i wmi tfregh akid iryazn nsnt nigh i wmi udvnen tarwa nsnt... d tidet, tugha tslan di tnggura nni mmis n Badi issawal g imrabdven, itini xafsn blli nitni d iwdan amacnaw nccin, ur gharsn tizmmar ad wcen ula ad ksn... d tidet, yusa sn d wawar nni yffegh abrid; maca ur days tiwiyn: mmis n Badi yffegh ighra cway di tneddam, u d tighri nni is idvarnen azllif: marra inni ghar todu tighri tffghen abrid... u snnej i way, mmis n Badi issawal x imrabdven, ur illi x wmrabedv nsn nitni! Maca tugha din ca nnidven ur illi nican g sggwusa nni: ttbasva n Yamna el Xmmari d tsfifin n Mimunt n Srwan dwrent tudrusnt di tudrin d rswaq; udfnt-i- d ca n tsfifin nnidven, issawal daysnt yijjn s marra mayn ghars d tizmmar; awal nns jar taorabt d tmisvrit u tilivisiun. G wmzwaru tisfifin a tugha tilint ghar usrmad n tinmel wapa; akd wussan dwlen attvas n yinni tugha yghrin cwayt uca ffghen zi tighri ssaghen tnt; ar ami ur tqqim bu taddart ur t udifnt. Tisfifin n «ccix kick» (ammu i-s- ittini Badi) ksnt awal i iwdan: ur qqimen iryazen ssawalen x wnzvar, x tyarza, x tmeyra... gw ayrawn di tmzida, di tmghriwin, di ssdaqi, imcraz d initciten ur qqimen ssawalen: ksen asen awal «inni yghrin», inni itslan ghar tsfifin n «kick». nitni wapa i yssawaln: ssawaln attvas, maca x ca nnidven. Gw awal nsn i di tssvur tmazight akid taorabt tasnt –i-d attvas tawalin: “bidoa”, “cirk”, “kufr”. Akid wussan, zegw ami ur qqimen imughar ssawaln, tuzegh tala tazuggwaght (ur qqimen ihudriyyen ghezzan t kuy assuggwas am zic); aglmam n izizawen yccur d s ubryudv, aman nns ursudven, zegw ami ur qqimen fardven t imzdagh. Inni yzwvan ghar Ujmmadvin ssaghen sitirnat n waman, itawitent-i-d bu ukamiyyu zgw aggwaj, ggarent di tmzizdit itwaggn s tsima x tzghwin n tudrin nsn timaynutin. Irwacunen ur ghar illi bu ca n mmitsen ajmmadvin ssekkan tarwa nsen x wghyul ghar wghbal n lmxzn yiggwjn azyen n wass n wbrid hama ad ccaren tibidutin yuran zi zzeyt s waman (iqubac n traxt ksn, ur qqiment s uyt umjjaw messrent-n). G isggwusa nni ssntant tmgharin tinint «salammu oalikum»: zzat uya tugha ttinint: “oawnk” – “salammu oalikum” tugha ttinint iryazn. Mmis n Badi idwel ur iticc afus nnes i tmgharin: “hram”. D tamghart nns i yggin “lhijab” (tasghumbart) d tamzwarut: tugha timgharin teqqnent tahttvawt nigh d tapinat di taddart u dg iyyar, tettvfent zzays acnkuk nsnt. Gw uraren d tmghriwin tqnent tasbnect. “lhijab” n tmghart n Badi yddar azllif, itnnedv i yiri d ughembub, ar aghsmir d tammiwin. Akid wussan dwlnt tmgharin marra tqqnent “lhijab” amacnaw tamghart n Badi. Imara ca n yict ur tteggi, ssawalnt days tinnidven; tinint xafs : “ur telli x wbrid”. G isggusa nni i tbbupli Tutuh: ugin tarwa nns ad ssiwdvn sksu ghar tmzida ass n loid, amen ttugha iteg babatsn. Nnan as : “tamzida i tzit ur tlli n macca”. Tutuh ur s-t yisi wqrab, nettat i d ikkarn x hennas d yemmas teggent sksu x txsact d tmellalin, tccarent tazuwdva, teggent g wndu, ssekant-t ghar tmzida hama ad zzays ccen imzzvilla. Ass nni tqqim Tutuh tthraywa, ttnnedv i taddart, tarzzu mamec gha teg ad tssek tabqect nns ghar tmzida. Mani ghad tsnuffer nettat udm nns? Mayn gha tini i ymmas yiwdven mani gha nawedv marra? … Ur d tusi tfuct ad tghri, ar ami ttugha Tutuh tnettvar arrud nns, tggur tssawal akd ixf nns. Deg isuggwusa nni i twaysin izvra n ucarcûr n tmzida n tqrint – mani ttugha tawynt tmgharin tarwa nsnt hama ad ggenfan zi tqqrint: itwanna d Badi i zzaysn yggin aynnir i wydi nns. Mamma n Mmuh u Heddu, l (radj) n wmrabedv, ur tkki ca awarni umi itwacar warrud n wmrabedv uca temmut. 2002 Tawiza. .
