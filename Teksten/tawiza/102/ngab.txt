 uïïun 102, mrayur 2005 (Octobre 2005) Tamazight Idrim n biblghu Yura t: mmis n tmura nes (Libya) Nuri di ammas n wussan D nuffu affay n tidet tamekra Neffed n iffad n isway n aman Nedder neffed s tiblemt ansel tedsva. Tidet nzver dis am idrem n biblghu Ur nufet it di tamurt negh di wujenna Ass n dytca yukert si nnegh acenqu Di af iffad nnegh terni tlazva n tusna. Tazdit di teniri innnej di wujenna tuli D af tyblemt n anzver wudan syewlan Tuli, tuli ansi tywetv itri d ay terni D mammu yetteter s wullem yetcar iyghef kan. Iysewyl ameckan si nnegh d yezga yemlu A baba tidet di warra n amezruy tella Yerru fllas babis: iskerksun dis babik yeslu A mimmi ukrun aqercal, uekrun tlaba. Yemlu ameqran si nnegh tedit a baba Di ass n dytca tidet s wufs nnegh hatnari Yeru fllas: a mimmi ukrunrt ass n dytca D tvam tlazva si lejrad yetcu mendi. iysussem alemmas si nnegh d yemlu di manis wuddut af innatv, biblghu ass natv yummut tidet a baba netta assa d matta neggu dis nsemda tawtra si awal nnes s iles nnegh neslut. 2002 Tawiza. .
