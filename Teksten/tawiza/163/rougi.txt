 uïïun 163, ymrayur 2960 ( 2010) Tamazight Muhemed rrugi d wamud nes amaynu “isvemmidven n usefru” Yura t: Andich idir Yffegh d ijjen udlis d amaynu n tawmat muhemd rrugui s tutlayt tafranssist adu assagh “isvemidven usefru”. Amec nssen di tira inssij xaf sent t zi tewart n ufsar nsent deg ughemmis n Tawiza tawmat muhemmed rrugi, ifetteh u yarezzu attvas di tanettit nnes Tamazight, s uya deg udlis nes nigh amud nes ntaf day sent tiqessisin amecnaw (numidya, arrif tamurt inu, tamezgha nigh tanettit i yreqqen, tamurt inuba inu ) deg wamud a mani ntaf ict n tighit d tameqrant di tutlayt n mulyyir d viktur hugu, mani irifien ra d nitni snfleln u uran s tutlayt a idjan xelli amenni d tamaynut xaf negh, tawmat rrugi yessiwedv attvas n tebratin zeg udlis nnes, attvas n ighuyyan id itfghen zi radjagh n ucar n tmurt nes, zi radjagh usinef iteddar s wadday nes tammurt n arrif, Amud n «isvemmidven n usefru» yegg as tamssizwaret azrawi u amedyaz amqran mass farid muhemed zelhudv. zeg wur nnegh nssitim as i tawmat muhemed rrugi ict n ticri ghar zzat teccur s isruraf d usnflul d tcuni. i tawmat rrugi neqqar as marra ayd negh ayyyuz. 2002 Tawiza. .
