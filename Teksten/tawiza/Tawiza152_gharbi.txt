 uïïun 152, mggyur 2959 (Décembre 2009) Tamazight Tanfust: Tarja n arrimet s raxart nnes Yura t: Yusef Oazz Eddin*, Asughel: Said BELGHARBI Di radjagh n yewt n uyezzim yuggi ad iggenfa, am wussan n buhbel inu ineghden di tisedjya n zmann yeccuren s umarret yarezgen, ussan yessrawen s tghennant nsen di yesder umuzzar s taqsuhi nnes i yuggin ad tsala, ict n tuddimt n idammen dja tudum d zeg ict n tafekka (corps) tessummet tisi n taxxamt inu.. Tuddimin u gha war yedji unfarar?!!... Manis nec?.. Oini zeg ijj n wakud itnedraf di tisuyar n gha war yedji tefraz deg udmawen, twarigh t am tebquqin xef tefray n waman yarewyen, am ca n icarfa yeqqim icetv zi ca n uparpad n yejghulen, umi yusus ighes sadu usvarwet n izenzaren n ca n tarja d Tabitaggurt**. Tuddimin tmudjughent xef marra reqcuo n texxamt, ggint am waccaren n tissist (araignée). Taxxamt targha s wezgher, min day s itucrur s tidi, mghar arrimet nni oad tnessis s idammen am ca n tagherwast (sacrifice) tnezvwadv s tikma (douleur) n taghrusvi. - Manis d tusa arrimet a, d min teona radj nnes?!.. D wa d aseqsi iggigh i ixf inu, ma d ttidet min twarigh deg uxxam inu; nigh d ca n turja zi turjatin inu yemxumblen, dja oemmars ma ggigh ccum n ca di turjatin inu, maca rexxu day sent ccum n waressvudv?!... Smuqlegh di tiysit yuylen di rectu, twaligh tiggwdi trabao am isardan xef tefraz n wudem inu, ggigh am ca n ijj yettu ixf nnes di radjagh n yefran n arrimet nnes… nigh am ca n tiri tennufser zeg unecref n radj nnes; swingmegh iked iman inu; min onigh nec?!!... Ksigh d adlis ad day s gharegh; maca ssentigh tarrigh d marra min ccigh!... Iskilen n tira nnes msarwasen di tuddimin d itnessisin zi arrimet nni, d cwayt idwel udlis nni ura d netta d tikma tessur marra s tezwegh n idammen. Innedv zzay i wakud deg ijj n uraq war das d tteonigh, d ca n ixarrisven temhizwaren di twangimt inu kessin ayi ghar ragwaj n izumal nsen. Tesligh i ca n ighuyyan traghan day i: - Mani ghar affegh azyen n tarja inu, huma ad zzay s mmudvegh azyen nnidven?.. Deg ammas n tarja n ighuyyan, deg ijj n umadval imsarwas di tnaxt n raxart (affres de la mort) yesneddun deg umettin, yurmen s ruruf n tebridin, hima ad nawedv s isvuraf nnegh ghar ughlul (eternité) n tilin (présence); di taggant (la forêt) n iman nnegh itwacarfen day negh. Min day i yejjin tvbeptegh d netta t d arrimet nni yuggin ad tejjiwen s idammen zzay s d itudumen, tacigh axmi taxxamt inu am ict n tgharrabut teggur ad tawyent tiseffay n ayel, mani arekwsent tijdvidvin n waman ixf nsent; ttvawnt xnunsent s wafriwen nsent di tezwegh n tnedwin n idammen!!.. Ttucrur tirmect inu s tidi, ucigh iked tarjijat tesneddu deg ixsan inu; am usegmi nni war yessinen min iffaren di txabyin n imal nnes!.. Wdvigh xef ucar n tmurt, war ssinegh azedjif zeg idvaren inu, ittvef day i wannay i day i sweddaren, am wenni yarezzun deg ixf nnes xef ixf nnes di test n tmeddit war t yuffi. D cwayt axmmi d twarigh azenzar n tfawat awarni tburjet n texxamt inu, d yewt n uwessar ismuqqul di ca n qqundoat tmudvent di traccwin n ixemrawen nsent!.. Ighuyyan ssa d ssa, axmi n ca n temgharin nnuryent di lwacun nsent, ccatent ayejdur, trunt imettvawen ssuren s tikma (douleur) n uxeyyeq. Jar ca n tayyut tedla s icudvadv nnes timitar n rxezart inu; axmi djigh ffudegh deg ammas n tnezruft (sahara) di issezghel yejdi s tmessi n tfuyt, d mani uzghent marra taryiwin zeg waman nsent. Ma zemmaregh ad affegh ca n tutlayt zi ghar ssiwlegh iked raxart inu; i tacigh tudes ayi d, am radas ttegg arrimet nni ghar i di texxamt inu?!. Mmurdgh ghar s, tedwel am ca n ddarb n tuqqarqect… tesligh deg ugensu nnes ca n tililt (chorale) n uxeyyeq, d ca n ifassen war ssinegh manis d swizvdven carfenn ayi iles xef wawal, arrin ayi am ugnaw di zumment tghuyyitin. Mghar nec iyedjan iked arrimet tessur iked idammen!!... S radas n ict n tarja tbedd deg iman inu; d ragwaj n ict n tidet tghum s test n tmedditin, day i itgewwaden ghar usezgher n wehray n tisar nnes… War zemmaregh ad swinegmegh deg ixf inu ad beddegh d amaynu n tlalit inu sadu ijj n ujenna, ad tacigh ura d nnec ddaregh am marra iwdan, rexxu djfegh i ridaret inu, nnufesregh zi tarezzutin xef unaruz inu, d yekkin timewwa n marra timura nni!.. I twarat nnidven tesligh i ca n ighuyyan shenhinen deg ugensu inu: - Mani ghar affegh azyen n tarja inu, huma ad zzay s mmudvegh azyen nnidven?.. * D amyaru d AKURDI. **Tabitaggurt: awal a yus d zeg usemqar n umussnawi ameqqran Bitaggurt; i itwassnen deg iyyar n tafelsuft (Philosophie) tamedlant. (Yarnu t umsuqqel, Ajouté par le traducteur). 2002 Tawiza. .
