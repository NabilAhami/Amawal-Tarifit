 uïïun 166, sinyur 2961 (Février 2011) Tamazight tamssizwarut: tamukrist n tnttit di lmrruk (tafult 3) (La problématique de l'identité au Maroc - 3ème partie) yura t: mupmmd budhan tanttit d waçur (identité et race): yizzat nenna is açur ur illi d afräis (élément) di tnttit. nssn is takti (idée) yad ttnmgala (être en opposition, en contradiction) akd timdlitin (convictions) n waïïas n imdlasn (zi "idls", culture = intellectuels, personnes cultivées) imrrukiyn i tflisn is d açur (iri) i ytgucculn (déterminer) tanttit. di tidtt, timdlitin ad d agmu^ä (résultat) n idls aorab mani açur itirar yat n tmlilt (rôle) d taxatart ciyan. max, dix, nssaååaj açur zg uguccl n tnttit? acku tazdugi tagrudt (anblal, parfait, complet) n waçur ur tlli s umntil n n taãuri (aruggay, mélange) n içuran agnsu n yan n uvrf s imant nns. s unalkam, wi izmmarn ad issntm (affirmer) di lmrruk is ntta d amaziv niv d aorab, xf uswir n waçur nns, 100/100? man (matta) tnttit vad yilin vur ipnjirn i vad d ijj umaziv yiwln taorabt? ina (wid) ayt bab n tamiärant (conception) taçuryant (raciale) n tnttit ttinin is tanttit n ipnjirn d ilulen s yan n iwil iããuë (mariage "mélangé", mixte) ad tili ula d nttat tããuë u tuggut. d wa d akmim (impasse) mani tssawaä tamiärant taçuryant n tnttit: kuy avrf ad vurs ilint timiäiyin (centaines) n tnttitin vad ylin d analkam (conséquence) n umsiwl d taãuëi jar aïï^as n twacunin (tiwjjitin) zg içuran uggutn. agmuä is ur ttili vur uvrf ad bu tnttit itwassn zzigs. arni var manayd is awank iqqn ad issavul (aus sens de "refléter") tuggut n tnttitin n uvrf nnes u ad yili, dg waddud n lmrruk, d amaziv, d aorab, d andalusi, dafriqi, d aspëawi, d uday, d afiniqi, d arumani... u ayad issummuk is awank, zund avrf nns, ur vurs bu tnttit itwassan zzays. imala (mc) açur ur isguccul tanttit, mayn xf ntfulla (s'appuyer, se baser sur une raison) puma ad nssntm is tanttit n imrrukyn d tamazivt d ictn ur digs tuggut? ntfulla xf tmurt n lmrruk illan d tamazivt zi lbda u d nttat iticcn tanttit nns i iwdan days izddvn mqqar ad ilin içuran nsn ggutn u tqnen vur waïïas n içurn ur llin d imazivn. tayunt (unité) n tnttit tamazivt n lmrruk ur ttsba (nier) tuggut ta^çuryant, tadlsant u talsyant (linguistique). maca qao tuggutin ad iqqn ad ilint tnafant (servir l'intéret) tayunt n tnttit tamazivt zund min illan di qao timizar n umaäal mani ntaf tanttit n uvrf d ictn acku d tanttit n tmurt mani izddv, akd uvzdis n tuggut n içurn d idlsan d ilsawn. tanttit jar untti amuvrabi d untti amaziv ((amsäfaë dg uïïun d iddan) 2002 Tawiza. .
