 Uttvun 77, Tzayûr 2003 ( 2003) Tamazight Dda Lmulud Yura t: Asiram (Libya) Di taddart n salem n ayt Moemmer, di tawrirt n mumun s (ayt yenni ) Ilul Mulud Moemmri. Yemmut ass n 25 di furar 1989 di Oin Ddefla 140 km in umalu n Dzayer. D nit gres 71 di loemr nnes. Yefku dda Lmulud Tmeddert nnes i tamazigt, d in imal n imazigen.Teffegh d tira, teffegh d tegri, yeffeh d aselmad (lmuderres), yeffegh d anelmad (Ttvaleb), yeffeg-d agerbaz (lmederset) n tamazigt . Teffegh d tufut, tuzzel tafat af tamurt. Ad Lmulud akuc a cek yerhem, maca gerneg teddred… yedder awal nnek lli tessufghed s idles n tamazigt, yedder asekkil lli sis turid, yedder ul nnek d iwdan n tmazigt. Tira n Ad Lmulud s tmazight Di 1951 yessuffegh d amawal: (tamahaqt - tafransist ). -Di 1967 yezdukel idvrisen n temsirin d yettak di tesdawit d'adlis uggun. - Tajerrumt n tmazight s tefransist. -Di 1976 teffegh d “tajerrumt n tmazight” s tmazigt. -Bulifa d Mulud Feroun snetlen di tattut kra n isefra n si muhend umuhend. Dda lmulud ikemmel abrid-nnes d wel yeggu-c idis lli wel yewwi-c sis, d may ukkud yeggu sisen. Dga di sugges n 1969, izdukel-ten d iterjm ten gher tefransist d adlis uggun. -Isefra n si muhend u muhend. -Di sugges n 1980, d yerni adlis: -Isefra iqdimen n leqbayel. -macapu tellemcapu: -tsisaw n imeckanen, amud (recueil ) yerr-et-tent in tefransist. -May yemlu dda lmulud af tmeddert n ccix muhend?: Ugurun reboin isuggasen af tmettent n si muhend u muhend, di 1989 D adlis isem-nnes: -Inna yas ccix muhend. -ahellil n tgurarin) Di sugges 1970 dda lmulud yewwedv af tmeddert d yensayen-nnsen s umata, d yessakez may nit ahellil, d mammek iduggel, d yerni in ayuh tamedyazt-nnsen. D yuha yezdukel-ten d adlis, d yeffegh d di sugges n 1980. - Amawal n tmazigt tatrart. Yuri-t dda lmulud di sugges 1974 . -D amawal-uh yemlayem s tantaliwin n tmazigt lli llanet d' agafa n afriqya. -Yella awal atrar yemsuddem sisent… di tasgunt-uh n imenza nessemres tawalin titrarin s (AMAWAL N TMAZIGT TATRART). 2002 Tawiza. .
