 Uttvun 86, Sedyur 2004 (Juin 2004) Tamazight Ibriden n wrum Yura t: Mimun Amsbrid (Biljika) Kker ad narri! Nec, ijen wawar, cek nigh d cem, ijj n wawar; ghari, gharek/m, huma nessoufgh it ghar yijj n wbrid. Nigh huma yuzegh unessis n wawren. Ca n twaratin, war ghari itiri min ghad arigh… nec tixsegh ad arigh! Tasent ayi d tira d timezwura, ixf inu yeccur s wawaren, maca war ghari min zi ten ghad ghensegh aya ghar uya… Ad ten d yregh amenni wapa mbra tasghunt n ca n yizri nigh n ca n tnfust, amacnaw tidmamin n tseddect nni iqedvwen, ad as tinim min yughin wa: ma yetjifjif nigh?!… Uca Kkaregh tmudvegh-fessyegh… Ass ad, tugha ggigh ur tarigh ca… Nec gguregh aked wbrid, nec ad acigh aked yijjen sin n wawaren mghar tmudjughen dayi: imicar n wrum!… Iwa min ghad ggegh? Aqa itxissva ad zzaysen sarwegh firu n ca n tnfust bac ad ten days ghensegh, nitni d wayraf n wawaren i dasen itenndven di tectawt inu! Iwa min tennid? Ad kidi tarid?… Min xafek/m d ssdvaran imicar n wrum? Mani tzeddghed, ad ac/m inigh! Deg wdrar nigh di temdint? Imara deg wdrar, aqa twarid min xef ssawaregh… Ca n tmeddit n wnbdu…Tezzredv xf worur deg wndrar, xef taqqa; ancer n wrum xef ubadu n undrar ytissiq adu tfawt n tziri… Iwdan ttvsen, advu yarsa; ajenna yudjegh, yiggêj, d azeyza ca n tuzzeyzewt tfsus, tudfit tmrer; itran anict n isvnnajen; jar ytran imuzzuren, itran izzdaden amacnaw taqqa n wrum ccuren ajenna… Tittvawin ggur ad qqnent, tsêkkdvedv deg wjenna, ttarjid ajenna… Ajenna yebdva deg wzyen: yebdva t ijj n wbrid n yetran, ca d amuzzur am urum, ca d azzdad am taqqa d tizzin, zi yeffus ar zermedv: abrid n imicar n wrum! Amacnaw abrid itawyen ghar tyart deg wnbdu: yessu s wrum. Tiracciwin n wrum, ict tdveffar utcmas, ssuyusent rum awarnasent, yetghima yegga yabrid n urum: d abrid d ijjen, zi tmurt n tiyarza huma d adrar, mani djant tizeddagh. Abrid n urum di tmurt: abrid n wrum deg wjenna! Di tmurt, abrid n ayt bab urum; deg wjenna, abrid n imicar n urum. Ajenna d tisit n Tmurt, Tamurt d tisit n wjenna! Tittvawin azyen-qqnent, jar wabriwen, ttwarid imicar n urum: ticri tfsus, isvuraf d imiriwen, ijjen wdvar itawedv car, ijjen war t’ itiwedv, ighadjen nsen yemmuden ttvfen di tracciwin udvuf n udvarghar; tighessawin n yeryazen d ikidvaren tazrent s tidi; awar war yedji: war ttsrid mghar i wsudvi n wanzaren n ikidvaren d tusut n yeryazen: imicar n wrum tmarran ad awdven huma war d xafsen teggar tfuct. Imara teyred xafsen zzat ad awdven, ad ten tessefsi, nitni d ikidvaren nsen, d tiracciwin n wrum: ad tafed ca war teggin: ad tafed amxmi oemmars war ccuren tiracciwin, war tent ghnisen, war tent zeyyren s tmezwarut d tmeggarut huma war twettvint xminni taryen tsawent nigh ttvaren taysart… Tiwecca, xminni ghad tyar tfuct, xminni ghad irin iwdven nigh fsyen imicar n wrum n ujenna, ad kkren ayt bab n wrum n tmurt…War tmsehriten, war temsezrin, war tmsiwidven; maca ssnen ayawya zeg ibriden n wrum yetghiman awarnasen: deg wjenna ghar yina, di tmurt ghar yinin. Di tmurt ad, ca yeqqen ghar ca, ca itemsvawadv aked ca, ca d tisisit n ca, ca d tiri n ca… War arezzu ad tebdvid huma ad tessned. Imara tebdvid, imara tqedvwed, imara tfsyed… war ttsned! War bettva jar ujenna d tmurt d yewdan d wawaren: da, war tzemmred ad tessned udem zi tisit… Jar wabriwen inu yettvsen, twarigh ten deg wjenna, aked wbrid n wrum: baba, s whfad nnes n tinghi itizzaren-yetyuran aked isvuraf yetmarran, atcamer yudef deg whfad, iffassen dvefsen ar taghmart n wghir, yezruza azedjif nnes zi twarat ghar tennidven huma ad yeghdver tidi, bac war as ttidef di tittawin; s ufus azermadv yettvef di tracca, s wenni yafusi yeccat asrrankem n tsardunt nnegh tacemratc ca n tiyyita fsusent: _ rra a yedji, rra! Badi n Hemmu awarnas, itmughzar aked tracca nnes: asardun nnes d azeyrar, netta d amudrus, d aqudvadv. Yessawar ghar baba; baba war xafs itarra: yessen billa Badi itexs ad yessiwer uca yessawar, wardji yexs ad yini ca! Hemmadi n Qeddur, awarnasen ghar aggwaj; yessapnaci deg wghyur nnes abarbac, ict n twarat ad yacem deg wghezdvis, ict n twarat ad as yewc ict n tighrit ghar wddis: rra a yamareymu ya! ma neccin ad neqqim d imeggura n yewdan marra nigh?… Ad yessudv wghyur n Qeddur s wanzaren, ad yeg amxmi yuzzer cwayt, maca war itekk ca uca ad yettu, ad yedwer ghar ticri nnes yezzupnen. _ ayenni ghad tegged!, ad as yini Qeddur. Ad yessek aghir nnes xf wghmbub: tidi tteg akids ighezran. _Rra a wen i dayi gha yenghen! Baba yessen billa Qeddur war yedji di ca. Xminni d as itarra taynit itarezzva d ghars: Qeddur das ur, ad xafs tekk ca n tijnewt, ad anegh yemmet aked wbrid… Abrid n wrum d tayffart d ict n tracciwin, zeg wzghar ghar wdrar… Rexxu twarigh ten, jar wabriwen inu yeqqnen: d nitni: Qeddur, Badi, Baba, Mmuh u Oollal d marra innidven, s isvuraf nsen imiriwen, yefsusen, s ighadjen nsen yemmudven, s ihufad nsen n tinghi i di yetsudv wadvu n ticri yetmarran… twarigh ten s nnej inu, deg wjenna, aked wbrid n imicar n wrum. Tsrigh i tusut nsen yudjghen, i wsahrat n ikidvaren nsen… Yekka xafi yidves, msagaren wabriwen n tittvawin inu, yemsagar wbrid n wrum n ujenna aked wbrid n urum n tmurt, fsyen di tirja inu n unebdu.
