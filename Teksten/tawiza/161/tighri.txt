 uïïun 161, tzayur 2960 ( 2010) Tamazight Tighri dg udelis n “Asfidjet” Yura t: Mustapha KABBOUTI Asfidjet, d ict n tesmudvt n tenfas tiqudvadvin, yura tnt umat negh amari Saoid Belgherbi. tinfas a tsulent tghimant zund ughun ghar asennadv d yimal, d min yeddar, jar asen, ufgan amazigh. Tinfas a, ntaf daysent attvas n tefras n iduran n arrif, am ssawalent x unamur arifi d igwrramen, x tsednin d warrvaw, x ighezvran d idurar, x tirrugza d tiggwdi, x tdvessa d imettvawen, x tudart di yarrif d ujemmadvin, tammurt n wezwag. mayemmi aya? acku iferdisen a, d nitni ixemren tacnuoi n umezruy arifi deggw agh n tismi x tmurt am nnan: «aryaz itmetta x tmurt nigh x temghart». Amari Saoid Belgherbi di tira nnes ikessi d zi tenfas tizaykutin, ca n twaliwin ntaf itent d nitenti, zund tinni i d anegh talsent thennatin xmi x anegh tesdara test, ntegg ixfawen nnegh x usummet, ad anegh tawi tarja n umezruy war itetvsen. Am ntaf amari ittari tinfas s ughanim n wezrawi, u manaya yewca azvri icnan i tirra n tenfas deg udlis n wesfidjt. Xmi ghad nghar tinfas a ntaf amarir yetru iggwed yessawal s iseqsiten x tmurt n arrif, yisi yas s unezgum, yarezzu deg iyyar n tmezzazvart d tanbadvt yezzvu umhars deg ij n was, danita ntaf amrir itseqsa x taryazt i ttugha zi tefras n ufgan amazigh, rexxu tengh iten tarjijat d tiggwdi, am yenna umarir di tenfust (15): (tarvwa nnem a Dipya mmuten waxxa ddaren, temmut ura d tarrvawt n wayraden… Dipya susef d tirrugza nnem d azawar i yinni cemm yettun d asendef deg iyezzimen n ucar nnegh azeggwagh), war zmaren ra d ssiwlen, ad inin neccin aqqa negh da, am nnan ij n was i Wsebbanyu, s ughewwegh d imenghan yarrin amhars am twuct deg ifri nnes, yexnunsen s tmenghiwt d wareyyec i yeyduden, tteffegh ghir di test, ma tufuyt ggwaren days ghir irifeyyen s yiri nnsen d azirar, deg idurar, deg ighezvran jemmaren bu yjarvwan yengha bu heyyuf s tiggwdi seg weyraden n arrif da ntaf amarir itawi d cra n twarritin x tghawsiwin a, yenna: «ad yiri d rebraqi n tneghda n arpaj iseyhen x idurar n arrif uca temmut daysen taryazt tendver kids tmettit d tlili», nigh d isefdvawen n tecmatect d tmezzazvart, i d itnufsuren zi taddart n taoraft watcmas n Yislam, da ntaf amarir yeqqarv di tenfust tiss (23) (belli aoraben arami d udfen ghar tmurt nnegh ttugha smidjizen, neszedgh iten kidnegh, nesmerc iten, newc asen ra d timgharin, maca arami harten smunan awar nnsen dwren d kidnegh s tiqqest neqqen anegh ul n tilawt nnegh yellan di tutlayt, di tusna d tdelsa d umezruy). iwa rexxu netru nkessi aghnja nessemlilih ufugh nnsen, ima d ffghen inni umi yessusem rezvgher d wesmuni n ttehrar d nnubet, qao timellalin n iyyazvidven n dcar smunen asen tent, i min ghad ssedren? awaren n refqi! yenna: Hram tasertit di tmezyida, mana wa tufid min gha tegged sebbeh. min ten yejjin ur zmaren ad ssijjen x turtit n umezruy ad ssaran cwayt di min zzvun imarvawen nnsen dayes, amezruy nni icnan i ghar yella watig amadvlan srin xas ura di Bitnam d yusin deg igiman n Wasya. Maca imazighen zewgen ghar tmizar n barra mani iswurin, mani ghrin sneflen tawengimt tamezrayt, ssenni xafs d ttraghan bac ad yekkar ad yarzu x min yeqnen ghar izerfan n tudart n waydud amazigh, zi tutlayt, zi tmetit, zi tanamurt zeg umezruy… s wemyawaz zi barra d ugensu iwdven imazighen ad reqdven amnad n tmeghnast. snsflulen ungalen, urin deg umezgun, setfen izran ggin imerqiten mani mma, sghuyyen x tesghart nnsen di mmarra min yeqnenn ghar tudart n weydud, da ntaf amarir yeqqar di tenfust tiss 16: ad negg belli aydud ameghrabi tesmun it tmazirt (awanek) yesmun it ddin macca ur t tsmun tutlayt d tusna d tawusna uzaydun tticcen imukkusen i wawanek, xedmen tammurt nnsen s ifassen d waccaren ktar seg waoraben, arami d nesmun marrva igerwan nnegh ghar tmurt nnegh maca war nesghidjet ca ighsuren n tzayzt, di tusna di tughmsa deg wermad d uremmud, maca uccin aked tizzvifin nnegh id itekksen izvewran n yinni yemmuten, iyejjin amezruy ra di kucci, ra di raxat, ma wa yenni wayrad n arrif «ad nemmet noizz war nteddar d imedrar». Min yenna wemghar n Arbatv di tinawt nnes ar idurar n Atvlass: «ad wem bnigh ict n tleggwit di Arbatv tmeqqur ad tesmun imeqrven nnwem ad zzveghren, waxxa yewt d wedfer am wenni yewtin di tunfit war ttebbin igharassen, iwa tepnam!!» ammu qqarven tarwa n xali Oellal afassi, ama zeoma tamzyida n lqaraweyyin tesmun anegh, tegga zzey negh ijj n weydud, ijj n arrvay, mana attvas n tmezyidawin ad msebdvan ura d ibriden nnegh, iwa mass Oellal gg arrihet x yinni wa ghar yelli wanzaren. Umat nnegh Saoid Belgherbi tinfas nnes marra tikkwra, marra di mettvawen ikessi s unezgum i imazighen mani mma, yarezzu x min yecnan deg umezruy ghar imazighen bac ad zzeys yarem tinfas nnes, huma ad qqiment d asendef i yini yexsen ad ttun yemmat sen, tayemmat id yesghemyen tagrawla d tirrugza, maca amarir ad yessghmi tissekla deg ijjar n twattut d useghdi x tilawt n umazigh. (Mustapha KABBOUTI, Tasdawit n Wejda - Tabridt n tghuri Timazighin) 2002 Tawiza. .
