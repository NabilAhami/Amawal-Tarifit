 uïïun 108, kuçyur 2006 (Avril 2006) Tamazight Asirem n tedsva Yura t: Salah Ngab (Libya) Isligh assnnatv yullul amdvan Isligh usunet tsilatin Adezregh ur ufigh ghri tettvawin Ademlegh ils nnu iskilin yenghu. Id itven imdan ussan Gher yutva ugregh s tmghrawin Zunun fus nnu imligh tinnigin maca tmedjit awal n yidi teslu. tedwl nnu am asirem tedsva dwlegh am ilem in tawalin ased memmek cek memmi yemla ittigh af abrid tira n tikliwin wekrun si si adinan d imlun icna d imluned awal n inzi tinnigin. asafar n attvan nnu kan tsilut maca mi ghessegh island negh tamettvut tsilut nnu tagherma nnu timum ad ada nnu dis yal ibucilin yal tbucilin. talji issalegh dis urwir n tufut maca id itven inni awerragh yummut tkerkas di tidet kan teslum teddrum iderghalen gherwen kan tettvawin. 2002 Tawiza. .
