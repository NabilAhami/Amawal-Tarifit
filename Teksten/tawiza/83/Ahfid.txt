 Uttvun 83, Krayûr 2004 (Mars 2004) Tamazight Ayyuz i tmesmunt n “tanukra” di Nnadvur Yura t: Hassan ahfid zeg Waliman Marra nssen ira marra regnus deg umadval ghar sen asggûs amaynu i zi twasnen, i zi tadfen ghar -deg-umezruy. Asggûss amaynu nettta d tfawt n weydud mamec mma yegga, d ij n rimart nnes i tejja t awarnas, d ij n tabburt-tawwart- i zi itadef ghar umezruy. Ra d imazighen ghar sen asggûs amaynu nsen i zi yudfen s yiri nsen d azirar ghar umezruy, asggûs nni umi yarnan imazighen -adu taryast n ugllid amazigh Cicunq- ifarounen di tmurt n Misvra 950 zzat i tlalit n Lmassih Oissa. Tamesmunt n Tanukra n tusna d usemghar di temdint n Nnadvur war texs bu ad tasseodu tafeska ya tamqrant mebra ma ad tegg rfadj d tcuni i wseggûs n imazighen, itemquddan aked usggûs wis 11 n trurit nnes. Tugha tarezzu ad teg ij n wurar yemghar n tafeska ya n usggûs amazigh 2954 di remrah n wuraren-Tiseghnas- maca ugin as ayt bab n tanebbadvt n Nnadvur-mebra m ad inin maghar ugin? Manaya itghima d ij n useqsi yemghar nesrusat??? Tanuka tzemmar s yiwdan nnes ad tssvurf tioenkrifin a marra, tzemmar ad tuyar ghar zzat s yiri nnes d azirar, u tiwedv huma ad teg urar a ameqran n imazighen usggûs maynu, s yijj n wudem icewwar attvas, di Sini n Arrif –Nnadur-itugha yeccuren ar aqmmum ass n 10-1-2004. iwdan i d yussin ghar tmesmunt n Tanukra tugha yen d iwdan ksin akid sen amnuss nican i tmazight, dvfaren marra iseddan n reghnuj, d umezgun, d tihuja zi twarat ghar tenniden, s rafrahet d ubarrep, d useqdec. X ughazdis n tafeska ya tameqrant, tegga tmesmunt n tanukra asemghar i weydud -iwazal- n snat imeghnas imazighen: Oellal Cillah anazur ameghnuj ameqran, d umeghnas amghar n tanukra mmas mustapha ben Omar, mamec tugha twara ya ameghnas Naser d aryaz n usgûs n 2954 am ij uqadi yemghar x tizmmar nnes i teg x wudem n tmazight. Ghar unggar tssutar tmesmunt n tanukra bac ad yiri ass n 13 yennayer zi mkur asggûs d ict n tafeska i marra imughrabiyyen am tifeskiwin nniden n unamur amughrabi. Asggûs amaynu d aghudan iffulki i marra imazighen d timazighin d yiwdan marra n umadval itexsen tirlli d ttisqqar d izerfan n utarras. Tudart i tamazight di tmurt n tamazgha xsen nigh war xsen ayt bab n wesoarreb. (Hassan ahfid zeg Waliman, ass n 8-2-2004-2954) 2002 Tawiza. .
