 uïïun 134, sdyur 2958 (Juin 2008) Tamazight tamssizwarut: asfugl n tktut n usnti n fas ur itmmçlay xf usfugl n tktut n tamstna n fransa xf lmuvrib di 1912! (La commémoration de la fondation de Fès ne diffère pas dans sa signification symbolique de la commémoration de l'anniversaire du protectorat sur le Maroc ) yura t: mupmmd budhan issnta usfugl (célébration, commémoration) unãib (officiel) n taktut n tskkiwt (construction, fondation) n ivrm (ville) n fas. anabaä (gouvernement) isngmä (réserver, affecter) yan walzaz (budget) n 350 imlyunen n udrhm puma (afad) ad yili usfugl dg uswir (niveau) n tasartut (place, statut) i var fas. tasartut ad mani itmun umyey (mythe) d tidtt, invlan (imaginaire) d tilawt (réalité), ijjd (religion) d tsrtit (politique), tadugamt (symbole, azamul aussi) d timsirin (événements). tamdint n fas ur vurs xzzarn zund ca n tmdint d tmazivt, skan (fonder, construire) tt imazivn u tuvat d tamanvt (capitale, tazuççnt aussi) n tgldin timazivn. mr asfugl ad d asiddr (le fait de faire vivre = commémoration) n umzruy amaziv n fas, ili nenna is asfugl itmsasa (convenir, s'accorder) akd taktut n usnti n fas. uhu, fas tavul dg umzruy unãib d tamdint tamzwarut i iska idris aorab i d iddan zi tmurt n waorabn dg ugmuä (orient) aorab. akd awaä n idris vur lmuvrib, issnta awank (état) amzwar itwasnen dg umzruy n tmurt a. d wad i min (may) ittini umzruy aorab. maca ur illi d manayad i d anv isnçmayn (intéresser, préoccuper). tsnçmay anv tumayt (contenu) d wammak (signification) n usfugl acku issummak (signifier) tagdlt (exclusion) tasrtit n tmazivt d imazivn. asfugl ad, dix, issnfala (exprimer) tumrt (joie) n uvbbi (extermination) n tmazivt awarni idris i yarrin lmuvrib d yat n tmurt d taorabt. asfugl ad isvlal (commémorer), dix, tamstna (protectorat) n waorabn xf imazivn. s uyad ur itmmçlay (différer, être différent) xf usfugl n tmstna n fransa xf lmuvrib di 1912. dix, asfugl zi tktut n tskkiwt n fas d yan n asming (glorification, tawlvi aussi) s tahrst (colonisation) taorabt. amija (danger) n inaw (type) a n usfugl is issvna (créer) yat n timkti d tamazrayt (historique) tlluxca (déformée) u tetwarjn (falsifiée). acku amzruy ntta d abrid i tawyn vur tnttit (identité, tamagit aussi). s unalkam (par conséquent), awddar (tuzzla) n ubrid d tafada (condition) tazwarut i uwddar n tnttit. s uyad asikz (reconnaissance) n tmazivt itettar asvti (correction) n umzruy nnv amaziv d ufran nns s qao isrkisn d umyiyn zi t ccurn aorabn. 2002 Tawiza. .
