 uïïun 157, smyur 2960 (Mai 2010) Tamazight Tanfust: Jar twizext n tidet d tarja Yura t: Said BELGHARBI D awardi i: “D asvudvem xef tenyart n jeddi Bezzah waxxa yemmut, d tenyart n henna Tapnuft waxxa teddar”. Wami d anegh ksen aghrum; war nenni maghar? Wami d anegh carfen di ticri nnegh xef ibriden nsen immexsen isvuraf nnegh; war nenni walu? Ksem anegh marra min texsem, jjem day negh mghar tufganit nnegh, nigh ura d tenni teksem tet zzay negh war akid s nerpi! Maca tuca akid s “henna Tapnuft” wami tarzvem awezwiz nnes xef tefray n uyezzim inu, txeyyeq; izvemm wul nnes; tuzzel xaf s tzvwa d ghezzu nwem; d wenni n zeldv ghar ugaji, wami innejfer umettar zeg unedra n dcar, wami tzvum “tara n trussi”; teswa imareddiwen nnes.. deg akud nni i di tesmun henna ibehrar ghar ticri nnes; teghnes tinitin n ucar d afeqqus i wul nnes; huma ad tewoa mani djan waman. “Jeddi Bezzah” war yuffi tisira zi ghar yuyar, ihuras n warri ighens iten s ifiran n ghumma n tjarrarin, xef tifednin nnes temxumbel tansa i ghar yettvef d ticri ghar mani mma. Test n tmeddit tessusem xef tazeqqa n taddart tameqrant n ayt n “Umarzuq”, mani munen i tzeddight marra lwacun nnes, issufegh d jeddi Bezzah tazyawt ad yacar zi tesghart nnes, ixbec xef uqemmum n tesraft, iskeff d marra min day s itwaxemren n wagra, yarbu t xef uorur n tsardunt, iymsebriden izzenz min yezzenz, ura d tasardunt bedden ifadden nnes zegga tugha wdvan, tessens marra tamedit d nettat teghzaz; tarra ghezzu n uzughmet n yarden i tugha tezuyyet d macca n ijj n wass, tarwa n Umarzuq icciten bu tsira n tadvuft; adan nsen qessen; qarsven; mselghaghen iked ayawya nsen, d nitni tettaren anzvar xef iqemmumen n tesrafin nsen yuffsen s wagra!... Jeddi indep d henna Tapnuft zeg ufuss nnes, issenyi t xef tbarda n tsardunt, yenna s: - “Abrid nnegh d ansa u gha war yedji unfarar, ndep ghar mani mma twarid tifawin xef tmewwa n izvughar, zzat ma ad anegh d idvfar Umarzuq d rwacun nnes ad anegh nettyen deg undvel n tesraft i nucar d reowetc i tarewra nnegh”. Tasraft tetwacar, Bezzah yarwer, teqqim tewwart n tezrift tesbixix, tayennurt tecca ighghedv n tmessi nnes, ijajjen n uxxam temsawadven jar ayawya nsen s tutlayt n utvebpet, ffghen d am ixedmiyyen deg wul n ca n arrimet temmexs aghimi nnes jar lazv d tyawant n tesrafin. Di tuffut n wass nni; isbuori wi isghuyyen xef tukkardva n yarden, maca attvas ggin ghezzu di lwacun n Umarzuq, wami adan nsen temghezzan s lazv, agra nsen itwanti ghar wazzay n taddart!. Jeddi Bezzap iccat s teghgharct nnes taqsuhi n ucar ifezzaren s tzawa; itleplip s tumart am wenni i yuffin txabit n wedrim, afuss nnidven iccat zzay s di zoenquq n tsardunt huma ad tefsus di ticri nnes. Tennas henna i yettfen di svrimet huma war zzay s tnejfir ghar unedref: - Ghar mani netgaj a Bezzah, aqqa tacigh iked uyenni itazzel xaf negh, maca acar nnegh isriwriwe d ghar negh, war yesxi ad nemutti? - Negh taycint deg ixf nnem, mermi mma teggid asvurif ghar zzat ofess xef wul nnem, rebzv it huma war itghimi iddar jar takkayt nnem!... Teddu; tesriwriw i mkur afruri d itgarrao usvurif ghar tawada nnes, teccat deg wadjun n wadvu asvemmadv izzelfen udem nsen izzuyten ijj n imal d amaynu, yenna s jeddi deg idv nni n uskuti di yesummet ict n txancet n yarden: - Imal nnegh iweddar jar tibadutin n dcar, neffegh d ad xaf s narzu mani nnidven, mani war nessin!... Iswedv di tmughri n temghart nnes, yuffi t tarekkwes rxezrat deg usqar n itran n ujenna, yareqqen jar tittvawin nnes, am wareqriqen n waman i tugha d itnessisen ijj n wass zi “tara n trussi” i yedjfen s ughir i waman nnes; tedjef ura i jeddi d henna... - Di mindi txarrasve d a Tapnuft inu? - Deg ass n udwal ghar dcar nnegh, akud di gha d smunen lwacun n Umarzuq ayt n uxxam; ad barrhen xef tadewla n Bezzah aceffar!. Iked usfarnen, yiwyi t unudem jar twizext n tidet d tenni n tarja, amarret n webrid issizgh it zeg idvaren nnes i yudvnen s ticri. Deg umnadv n tmazirt n Lgherb; mani i yiwedv, tibehhar n Ufasi msedjaoent; ssunt d agra; am tidekwanin n jennet; refrug n tayyutin dlin iyran nni ugha war yedji ufray, taryiwin n waman tazlent am ifighran xef tifednin n imezdagh, xef idvaren n henna d Jeddi d tsardunt nsen, uylent ghar udmawen n kradv idsen timitar n zzeldv d tinfas n uneoreq d uxeyyeq, Afasi issen ira usin d zi arrif, zi tmurt n lazv... Yeksi d henna di turvut n ufuss nnes am waman drust, isghim itt di tiri; tusmin n jeddi war jjint tamghart nnes ad tadeff amadval n tawuri, netta ittar zeg Ufasi, ad imjar nigh ad icarez, huma ad yarzu xef ughraw zi ghar issenta imal nsen. Afasi isgha xaf s tasardunt d min d as icetten n yarden, ittar ad xaf s isegh ura d henna Tapnuft, maca ikkar d zi tarja nnes itarjiji itemzawar; iteghzzaz tighmest s ufeqqeh; itmengha, yarezzu xef uxedmi zi ghar inegh amcum nni n Ufasi, yeqqar i henna u nettat war tessin min tid icqan deg idv nni n tayyut: - Ewa d ayenni i dayi itghiman rexxu, mara war iwyegh i babas tamennawat ad taff nec war djigh d Bezzah, yarezu ad xaf i isegh min war itmenzin; arri n ugarrem yexs ad cem yarr ghar s d tayya!... Tapnuft tewca s ad isu aman zi ghar issexsi tarjin n isefdvawen n tarja nnes, di war yesxi ad yacar tayri nni waxxa war yedji deg usayes n tidet. Nec i ytzemmamen tanfust a; righ ad awem inigh, sin n ineghmisen zi tmeddurt a: - “Itcettih wenni yennan jeddi d aceffar, maca itwacar, tcemtit raxart assegwass n 1934, hedd yeqqar yiwyit ayl n Tanja, hedd yenna tennujud ghar s tmettant deg idvesv wami tugha i tarja min xaf s urigh, jar ta d ttin; dinni ict n tidet; d anfarar n ict n taryazt d ijj n umezruy. Henna tezvguttvi day s tewsar, immexsit usvurif uca tewdva!.. tarni ticti ghar tiyyita nni day s d abarru, teqqim tcett; tmuttuy zi rectu ya ghar win, yudven ighess nnes, tedjef i ticri nnes, rexxu; aqqet d tarzift jar ifassen n wakuc, d yenni n yedjis”.
