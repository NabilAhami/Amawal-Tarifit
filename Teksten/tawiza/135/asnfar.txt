 uïïun 135, sayur 2958 (Juillet 2008) Tamazight Tanfust taqudvadvt: Asenfar n raxart Yura t: Said BELGHARBI Yesseghri dcar aweddar nnes, rami ijeyyef webrid timewwa nnes s tubbarcent n tizeft yeswin amessuci. Ayt bab n dcar ufin ixf nsen ggurn x tizeft, tghiren ggurn ghar imar nsen. Tizeft tedra amessuci, zi raggwaj ad tghired abrid am ij n ufighar d aqarqac n weoban, yezzver, yuss d ad yeqqes i dcar. Abrid am wa d asutar ghar marra, dcar itnufsur zeg ixf nnes, itadef tifawin, dcurat gha war djin igharasen qqnen x ixf nnes, tmettan deghya, ineqq iten raggwaj nsen x yebriden. Imezdagh iwyen d abrid n tizeft, abrid n tizeft yiwi d raxart ghar dcar, Tmammat war t yecqi webrid ura d wi xaf s yegguren, tennum akd toejjuja teksent ca x wur nnes, tennum tsvuruf x tebridin yuyren ghar tisedjya n dcar, aray am tseddyin yareqmen s tlullucin, yemmarsen x tedmarin n tmehtac, tibridin tawyent ghar mani mma, waxxa ttasio nsent anict n ratar n udvar. Abrid n tizeft yiriw, maca d aghufi x wur n Tmammat, ttaca yuss d ad xaf s yeqqen azzvri n dcar, rexxu dcar itqarqub s iqedduhen n tumubilat, ca ytari ca iteddu d, ccum n lmazutv yesduqma tinzar nnes, tesittem ghar idvennadv nnes ad yedwer, ad yass s umeddur n yewdan nnes, tugha yegguren ghar zzat, waxxa walu ibriden n tizeft. Yekkar uraghi s ighuyyan di dcar, Tmammat oad war tenfirir zeg uxarres nnes, d ict n tvumubil ddarnit tazra nnes ghar ragwaj n webrid, bab nnes yeswa deg idammen, yudef deg wuzzar, Tmammat deg afuss nnes tqabut n rehtic, trettex, tssenxuxur zzay s taqsuhi n wuzzar n tvumubil, yenghin axarres nnes x tma n webrid, abrid d yusin d asenfar n raxart ghar ihesyan n dcar.
