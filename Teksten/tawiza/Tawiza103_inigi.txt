 uïïun 103, ymrayur 2005 ( 2005) Tamazight Tanezruft n tgudvi ittadumn gh ighbula rufanin Yura t: Igellin Ussan zzugazn s mani yaggugn ar ghar ssan!! S tmedyazin gh kra n tusdadin n ifesti zund adfel ijdern, aggwent d wen nm iggi n tifert zvergh nn gisent tayelgi ddemgh ar sseflidgh i tadvsa nm. Kraygat as teffarri siti s wiss sa igenwan zund tatbirt tuzzvumt n umuddu n igdvadv, afgh km inn tezzeght gh uli nu, ddergh, ddemgh ddu n sa n ikaliwn. Hmelgh akal ann lli yi yusin d kemmin ihbu yi s usfulki. Ibidd ifessti ibughlan s wawal, uli nu ar ittergigiy tuzzvumt itteccbihn nm cuwwernin, biddgh nkkin ur ssengh izd ad kcemgh nghd upu, nkkin iqd iyi ughni s uzzal n tayri izzveggaghn, idvgam… iwwerri d!! nghd… S uswadur a azekka ixlan is rad bughlun ussan n imal? Gh tuzzvumt n izvula nk!! muccanin, tiwalin nm ad ibughlan s ijeddign swanin s waman n imuzzar, ar zzigizn gh izvuran nm, zziwzn km tuzzvumt n wadvan ghezzifnin. Izvula nm munen d ughrab n tudert ihman, muddugh gisn nkkin tusut f tusutin ar sdvuqqurgh xef ussan n tayri, negh tusdadt s tusdadt, amettva ifruri ijujg tuzzvumt n tnezruft n tgudvi idran, itaddumn gh ighbula rufanin, sekrent gigi nkkin taccumoit ittaddumn, s wadvu da d ikkan iffus, nkkin payi gh iggi n taccerjemt n tudert nm, ar tqqelgh s kraygat taghawsa. Nkkin gigh win imettvawn macc ur zvdvargh ad agh, ddergh gisn, gin iyi asmun gh teghzi n ussan herranin. Nkkin tigira ad ura sul syafagh inekkidn acku zzeghn dari s mennaw wanawn…. Yat tusdadt… ma s nnigh…. Nnigh masd nkkin ttukrafgh s izvuran n tayri, tudert inu tga tarragt taxatart i xti mu nnigh yan wass ra nmun zund tamunt n ughu d waman. Anaruz inu gh tudert ikemdv, ijder, ass lli nn gan middn tuggas ad kkin nger sin ulawn mhamalnin agwin ad bdvun, waxxa tekruzzv tudert. Tumert nm gh iggi n udrar n ddunit tezziwzi yi ghemkelli izziwiz wattvan amagus n ussan, tekrezt tayri nm gh ul inu, ghemkelli ikerz umekraz idverfan gh wakal idunkeln. Righ ad gim saqsagh «manik ad tgit mnid n tlalit n wussan», mac mayann yaggug fella ghemkelli yaggug uzenzvar n tafukt f wakal. Kemmin ad isslemden tayri, tegt tt inn tuzzvumt n wul inu. Tekrezt gis tumert n ussan nnurzegh ad km annigh, mac ssurf iyi, ad km ukan annigh, zverghkm, yamzv yi udupdu n tayri nm, izvula nm, azzar nm, igh ar tmussun, tesmaqqelt gigi, gegh aman n mrur. Tili tasa nu iggi n urfan n ufarnu, kremnin. Tabrat nm mutvlgh as, izd atbir da tt id ittawin is immut tuzzvumt n ugharas, nghd is ittut s lbarudv n inegmarn, tuzzvumt n ughras, ur d yad sul ilkem igellin, nghd is ka imutvl!! Ah! haqan is immut, fsin ixsan ns iggi n wakal, ig tawargit n tifawt. Izula nm igh ar tmussun, s wadvu n temdellest n wass, fsin ixsan n tuzzvumt n wul inu. Igh ad sn enn tgit tazvult, ar tbbin tifeyyi n tuzzvumt n ixsan inu. Atbir n tebratin lah t, ur d lkemn, ilih ineghmisn nm, gegh nkkin inigi n ussan ngh. Ahh inu, tesgeldit gh tuzzvumt n wul inu, ggammigh afsay i tmukrist ann. Iwerri d ad bughlun ijeddign, tuzzvumt n izvula nm muccanin, tiwin nm bughlant s imettvawn n tayri ngh, izvwighent, ur akkw igi ghikann tamyurt nsent. Tgit takiyyawt inu, yack id igidr ameghdar, abeldi, izugn, irufan gh wiss sa igenwan yasi km nger ifassen inu. (Yura t: Igellin, Agadir ass n 15/06/2005) 2002 Tawiza. .
