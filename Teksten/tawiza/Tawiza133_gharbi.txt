 uïïun 133, smyur 2958 (Mai 2008) Tamazight Tanfust taqudvadvt: Yumes ayejdur! Yura t: Said BELGHARBI Yuzvegh urimam zi tuffet n waman, itzvadja uzidjez x tmewwa tugha itareqriqen s inessisen n tnedwin nnes. Iswedv Mmuh aqeccar ghar ujenna, ixezzar ma war itwiri ca n tuddimt n wenzvar tus d ad tesgenfa arimam zi razagh nnes, maca ajenna izvum ura d netta x wudumen, yara iyyar nnes d aziza, tweddarent day s rxezrat n imezdagh. Mmuh aqeccar war yedji ca d aqeccar, maca tarwa nnes marra d iqeccura, ura d yedji s tamezvyant, rexxu turi d ad temghar, maca itari d akid s ura d tacruhi n ixef nnes, umi war yisi uzedjif nnes acewwaf am yessis n dcar. Txezzvar deg urimam, ttaf it iqquccar ura d netta, uca ad tezyenfa ghar awaren n yimma s i das yeqqarn: -“War temxumbur a yedji deg isekraf n tsarrawin nnem, aqqa oad ad am d yeghmi ucewwaf d tisedvwin, ad as neqqen rhenni, ad t nharred d rmepdvur, rexxu awi d azedjif nnem ad as neqqen asafar”. Ad as tames ticcart d yeghghed, huma ad ghemyen izuttvan nni immexsen izedjifen n tarwa nnes. Aghirin i wehbudv mani yejna urimam di tuzzvumt n dcar, din ict n teorurt ura d nettat d taqeccart, assegwass n tefsut n idemrawen marra sghemmayen rehtic, maca taorurt taqeccart tuggi ad tesseghmi ca x ucar nnes, tzvum ura d nettat, s uyenni tezidjiz axmi day s txezzar yedji s n Mmuh, ttaca ghar s ur am nettat. D ttidet, urawen nsent msarwasen, amen imsarwas umezruy nsent, Mmuh netta issen tanfust n teorurt taqeccart, inna yast jeddes yeccin ticti, iqqim itqidvar s uzyen n udvar, issuyara t d taghennant x ucar n dcar, umi tugha yeqqar: -“Oad ddaregh, oad day i buhber ad xaf ak uyaregh waxxa dvar inu yedjuz isvuraf iseggmen x yebriden nnec”. Rexxu jeddes n Mmuh yemmut, yeqqim zeg iwaren nnes mghar tanfust x teorurt taqeccart, yetvfen dcar huma war itrikwir ghar mani nneghnit, teqqim d tameduketc n urimam yeffuden aman, teqqim ura d tameduketc nnegh i neccin rwacun n dcar. Mmuh yisi day s s lgha, yisi deg izran, ad yini timenna tugha day s itwajeyyfen attvas zi mermi, yenna attvas nneghnit, mermi mma yeqqim ghar tma n urimam, tkessi t twangit nnes ghar iyyar n zic, ghar umezruy n urimam, mani tugha djan waman d agra uxef war tmenghin imezdagh, ass a imenghan marra x waman. Arimam n ayujir Isrughan timessi. Min tesseswed d tinuyam Tjeyyf t d tisessi Manis kkin wamn nnec Mmutyen ghar mani? Swin zzay k imetlao Teksed anegh afadi Yezuyyet day k wur Deg wudem nnec afaji. Arimam n dcar mechar yiwi n inuyam d tnuyam, mechar isessu n wurawen, rexu yuher ura d netta, maca Mmuh war yesexi ad zzay s ikkes rxezrat, itghima ghar s mkur ass i fetter d x ucar nnes tiqessisin, itghima aked imezdagh i iqqar asen min das yenna jeddes aqidvar: -“Ass a war itettu tifras n idvennadv, taorurt taqeccart ssuysen xaf s arpajj, rebraqi idduqzen di dcar d yenni d acemmudv yenghin ur n ucar n teorurt a, arpajj ssuysen t x umezruy nnegh, dduqzen x iyezzimen nnegh huma war tgenfin oemmars, xsen ad neqqim d imeprac n tcetca ghar sen s iwezwizen nnegh”. Di ltva n umcubbec jar uhewwas d irifiyyen, tugha irifiyyen trun x tmurt nsen, di ltva nni izi neccur irimamen s imettawen nnegh, ass a umi war netru x ca, irimamen uzeghen, ahewwas yus d ad anegh ider tudart nnegh s ihuras n raxart, oad yeqqim ccyadv n arpajj nnes i tfewwar d ghezzu x izedjifen n imezdagh, war sghemmiyen imuzar n wazvri, idurar nnegh war sghemmiyen i yemmumen n tcuni, irimamen nnegh sgheryen jeghdid, djfen i waman nsen. Mmuh yewta ayejdur, yessusef deg wudem awen nnegh ass umi yessen tanfust n yedji s temsarwas aked tenfust n teorurt taqeccart. (Said BELGHARBI, 28/02/2008) 2002 Tawiza. .
