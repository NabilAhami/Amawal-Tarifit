 uïïun 137, tzayur 2958 ( 2008) Tamazight Tanfust taqudvadvt: “Muci” war yedji d midden!! Yura t: Said BELGHARBI Tiseghnest n Trayetmas inxarrm as waghghraw s wattvas n weghnas. Teddu s tigemmi n Muci, huma ad as iodel min zzay s d iftuttzen n nnuqart. Muci d amezdugh, ura d netta d mmis n dcar, azvwar nnes yeghmi d deg acar di d ghemyen izvewran nnegh marra. Mara nenna s neccin d imezdagh cek tusid d ametluo, ad taf ura d neccin nus d zi manis d yusa netta, maca marra idnegh d imezdagh deg ijjen n wexxam. Yeqqim Muci x tpidvurt, zzat s arrabuzv, itsvudv zzay s deg wur n tmessi yarghin d arsawen, aweqqeh n isefdvawen issefsay nnuaqart, itarra t am waman qquren, tudumen di reqwareb, teffghen d d tiseghnas, nigh d ixerxaren temraywan x tceobin n tniba n dcar. Neqqen i wurawen nnegh ijjen n weghnas, di dcar ismun anegh ijjen n uzir, txemmer anegh ijjt n tmeddit, netted zeg ijjet n tara, necca anggur deg ijjet n tebqect... Trayetmas trehhef s uhewwaq i wudem nnes, tudef x Muci, ijja s amcan i wghimi, huma ad treqqem abzim n tseghnest s ufuss nnes, ifassen iswizan iwmuni d weodar hsen zeg inni iswizan i wardar n min imunen, min d anegh ifesyen ad t nesmun jar anegh. Trayetmas teswiza aked Muci deg uodar n tseghnas nnes yarzven. Yenna s Muci i Trayetmas: Ghensi t i rizar nnem, ad twarigh ma usint id ca x tcuni nnem. Deg wawar n Muci attvas n tighit yenuffaren, maca Trayetmas tettu tiseghnest, teghnes iwaren n Muci d tayri ghar wur nnes. * * * * * * Muci itaca aked ixf nnes d midden jar asen, d amsebrid, d ismegh, itarem tibardiwin, iteqqen ticedvyin i yeghyar nsen, d amzir itreqqam tiyarsiwin n isughar nsen. Itaca aked tesghart nnes tetwacc. Affegan umi ghar yemsar ammu s ughghir ad d yas ass di ghar yedha d agrawli, ad yarzu x tiseqqar nnes jar amzviwen yumsen raxart i tudart. Jar imezdagh d Muci tadukli ghar yedja udem n zic, maca netta yessen tidet n tuffrin n waydud di iyteddar, mani itaca aked tadukli nni tarecca, itaca din aymaren jar asen, jar as d Trayetmas i yugin ad temrec ghar Muci s taqsuhi n izerfan yudvnen di twengitin n imezdagh. Maca amuni nni yugha s tghennanin di radjagh n tughgherma nsen, amezruy yenna negh di tefray nnes, msemracen jar asen, msarrawen, aydud imeghghar x ixf nnes, maca rexxu tiriwen day s imenghan ugha war yedji wudem, imenghan i tjeyyafen tudart n tcuni, tcuni nni yeqqimen d asutar ghar Muci d Trayetmas. * * * * * * Yus d ukud n tzadjitin, Trayetmas tarra udem nnes ghar akkuc, tudef di rfadj n tzadjit nnes, Muci yarra udem nnes ghar akkuc d wenni, yudef deg izerfan n tzadjitin nnes. marra deg ijjen n useddi, d sadu ijjen n wexxam, d zzat n ijjen n wakkuc.
