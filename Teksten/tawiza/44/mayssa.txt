Ghvar ayi d awaren n tanust oad war turi! Meolik i tura iri war ntemweddvar ijjen x wennedvni. Ghennej ayi. D tisvurifin n wussan x webrid tedvhant d araji, X lgha n yiman deg iwed irettex d aseqsi. Tirmect imesr it ucar, rwehran x rwehran yarenni. Arrimet d ighedjen, tderreo i tudart a mayenzi? Ghennej ayi. D tisedvwin x arragub tharabent awarni tfuct teghri teghri! X tnebdat n tewwart i di tbedda ticri X ughembub n tmûrt i di tewsar temzvi, X yirar n tara i yesfadan tisessi. Ghennej ayi. Ttugh tirart inu, nghigh tûrjit inu x tefras n useymi.
