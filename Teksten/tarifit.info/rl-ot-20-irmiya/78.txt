Izři wiss arebɛa - Řmuɛṣiyyet n yeǧis n Urcalim

Alif. Mammec igga ibbarcen wureɣ ammu,
mammec ibeddeř wureɣ iṣfan ammu!
Mammec ttwabezzɛen iẓra n Zzawect Iqeddsen
x tɣemmura n ibriden!
1

2
Bit. Iḥenjiren iɣřan n Ṣihyun ttwaḥesben am wureɣ iqquren.
Mammec d asen tteggen am iɣarrafen n třaxt,
am řxedmet n uqeddar!
3
Gimel. Ɛad tuccanin n řexřa i-dd issuffɣen tibbicin,
ssuṭṭuḍent imeẓyanen nsent,
maca yeǧis n řgens inu tewɛar
am tenhaltin di řexřa.

Dalit. Iřes n weḥḍiḍ iřṣeq ɣar waneɣ nnes s ufadi,
iḥenjiren imeẓyanen tettaren aɣrum,
maca uřa d ijjen war d asen t yebḍi.
4

Hi. Inni tuɣa ittetten zic macca iředden,
ɣacan deg ibriden,
inni ittwasseymen zic x uzeggʷaɣ yuẓɣen,
ḥennḍen řexxu di řeɣbar.
5

Vav. D tameqqrant řmuɛṣiyyet n yeǧis n řgens inu
x ddnub n tendint n Sadum,
tenni innqeřben deg ijj n umur,
břa ttsaxur n ifassen.
6

7
Zayin. Ineɣraf nnes ittwaḥuzen tuɣa ṣfan x wedfeř,
d icemřařen x uɣi,
arrimat nsen zewɣen x řmarjan,
addud nsen yareqq am ṣaffir aziza.
8
Xit. Řexxu ibeddeř waddud nsen, idweř d abarcan x taǧest,
war ttwaɛqiřen deg ibriden.
Tiřmect nsen teřṣeq x yexsan nsen,
tuẓeɣ, tedweř am ukeccuḍ.

Tit. Iḥřa inni ittwanɣen s ssif
x yenni ittwanɣen s řaẓ,
maɣar nitni ihellufen am yenni ittwasnuqqben s wenqas n ṣṣabet n yeyyar.
9

Yud. Ifassen n temɣarin nni tuɣa ittḥinnan qbeř,
ssnennan i iḥenjiren nsen.
Nitni dewřen asent d macca,
zi ssibbet n treẓẓit n yeǧis n řgens inu.
