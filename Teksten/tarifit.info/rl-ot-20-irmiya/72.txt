Vav. Netta iqřeɛ s lɛunf aɛiccu nnes n ifetwan,
am ijj n weḥwic,
yareyyeb amcan n umsagar.
Issettu SIDI di Ṣihyun
ass n řɛid d ass n ssebt,
u deg wesɛar nnes
inḍar netta ajeǧid d ukehhan s useḥqar.

Zayin. Idweř Sidi inu icarreh aɛalṭar nnes,
inḍar Zzawect Iqeddsen nnes,
iwca řeḥyuḍ n yexxamen n řeqṣar
deg ufus n řeɛdu.

am deg ijj n wass n řɛid.

Xit. SIDI ixarreṣ ad ihdem řḥiḍ n yeǧis n Ṣihyun,

Ijja acbar n zzat d řḥiḍ ad ceḍnen,
aqa wḍan as ifadden ɣar temmurt.

Yud. Imɣaren n yeǧis n Ṣihyun
qqimen x temmurt, bekmen,
zzuzzaran taɛejjact x uzeǧif nsen,
beysen ixef nsen s txuncay n wecḍan.

takrict inu ttenhezza,
