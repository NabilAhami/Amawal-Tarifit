Izři n yesři x tesřit

“Aqa da, tecnid cem, tameddukkeč inu, cem tecnid.
Awarni i lltam nnem tiṭṭawin nnem d tidbirin.
Acewwaf nnem amecnaw taḥimart n iɣayḍen
arsin x idurar n Jilɛad.
2
Tiɣmas nnem ggint am tḥimart n wuǧi řsin tt,
ttgaɛɛaden-dd zeg wemcan n uṣebben,
marra tixeswin urwent tacniwin,
war teqqim ict nbřa iḥenjiren.
3
Iyencicen nnem am ufiřu azeggʷaɣ iḥedqen,
iṣbeḥ uqemmum nnem.
lḥenkat nnem am tremmant ibḍan
sadu lltam nnem.
4
Iři nnem am ṣṣumɛet n Dawud,
tebna d tisfifin n tseddiyin.
Ařef n tcappat i useřqi n řɛeskar uyřen ɣares,
marra tcappat iqesḥen n useřqi n ayt n bu-teɣruḍt.
5
Idmaren nnem ggin am tnayen n iɣayḍen imeẓyanen,
am wacniwen n tɣayḍet n wezɣar,
heddan di tlillicin.
6
Ař i ɣa ismeḍ wass u ad arewřent tiřiwin,
ad uyureɣ ɣar wedrar n umirru,
ɣar tewrirt n jjawi.
7
Cem marra d aẓri, tameddukkeč inu,
war dayem min i ɣa iɛeyyben.

Hwa-dd akidi zeg idurar n Lubnan, a tasřit inu,
hwa-dd akidi zeg idurar n Lubnan,
ṣwed-dd zi tqiccet n Amana,
zi tqiccet n Canir,
zi ɣar řmargaḥat n wayraden,
zeg idurar n iɣirasen.
9
Tucared ayi uř, a učma, a tasřit inu,
wah tucared ayi uř,
s ict n řxezrat n tiṭṭawin nnem,
s ict n tseddect n yiri nnem.
10
Mammec tecna tayri nnem, a učma, tasřit inu,
mecḥař teṣbeḥ tayri nnem x binu,
řefwaḥet n zzect nnem x marra řeɛtur.
11
Zeg iyencicen nnem, a tasřit inu,
tettudum tamment n teɣraṣt,
tamment d uɣi sadu iřes nnem,
uca řefwaḥet n warruḍ nnem am řefwaḥet n idurar n Lubnan.
8

4:5 ‘tilillicin’ - ijj n marka n inuwwaren daysen aẓri aṭṭas.
